        JsonObjectSchema itemSchema = JsonObjectSchema.builder() // see [1] below

                .addIntegerProperty("age")
                .addNumberProperty("height")
                .addBooleanProperty("married")
                .required("name", "age", "height", "married") // see [2] below
                .build();

        JsonSchemaElement arraySchema = JsonArraySchema.builder()
                .items(itemSchema)
                .build();

        // 构建根对象schema，包含一个名为"people"的数组属性
        JsonObjectSchema rootSchema = JsonObjectSchema.builder()

                .build();

        ResponseFormat responseFormat = ResponseFormat.builder()
                .type(ResponseFormatType.JSON) // type can be either TEXT (default) or JSON

                        .build())
                .build();

        // 问题描述

注：For OpenAI, the root element of the JSON Schema must be a JsonObjectSchema, but it was: class dev.langchain4j.model.chat.request.json.JsonArraySchema。构造的时候必须为Object，只能在Object里面添加一个Array属性
