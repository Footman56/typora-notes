深度学习是机器学习的子集。在机器学习中需要人工标注特征和数据处理，而深度学习通过神经网络来自动从原始数据中学习和提取特征

| 特点         | 传统机器学习                                           | 深度学习                                               |
| ------------ | ------------------------------------------------------ | ------------------------------------------------------ |
| 特征工程     | 需要人工干预：需要专家根据领域知识手动提取和选择特征。 | 自动学习：算法自动学习数据中的特征表示，无需人工干预。 |
| 数据         | 可以使用相对较少的数据进行训练。                       | 通常需要大量数据才能获得良好性能。                     |
| 硬件要求     | 可以在低端计算机上运行。                               | 严重依赖高性能硬件，如GPU，进行大规模计算。            |
| 学习方式     | 将学习过程划分为多个小步骤，通过合并结果输出。         | 通过多层非线性变换处理数据，实现端到端学习。           |
| 算法         | 基于各种算法，如决策树、支持向量机等。                 | 主要基于人工神经网络，尤其是深度神经网络。             |
| 非结构化数据 | 通常需要将数据预处理成结构化表格。                     | 能够直接处理非结构化数据，如图像、文本和音频。         |
