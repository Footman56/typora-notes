# 考核按钮

## 流程

### 自评

+ 自评等级不可见：员工页面无等级下拉框，如果是分数关联等级的话，还是会有对应的等级的，如果不关联等级的话，就没有等级

+ 自评 评分永远必填

+ 评语必填：员工页面每个指标的评语、综合评价里面的评语（同一逻辑）
+ 如果分数关联等级的话，等级不可修改，分数是自动计算的

### 考评

+ 等级不可见：员工页面无等级下拉框
+ 考评过程可见(**适用于指标和综合评分、评语**)
  + 仅可见自己考评：自评的评分、评语都可见
  + 仅可见自己及之前的考评
  + 可见所有
+ 评语
  + 开启评语必填：所有指标的评语、综合评语都必填（同一逻辑）
  + 不开启评语必填：所有指标的评语、综合评语都可以不填（独立于考评方式）
+ 考评方式
  + 整体考评：每个指标的评分、评语可以不填
  + 逐条考评：每个指标的评分要填



考核表考评

| 考评方式 | 结果                             |
| -------- | -------------------------------- |
| 整体     | 综合评语是必填、指标的评分不必填 |
| 逐条     | 综合评语是必填、指标的评分必填   |

指标考评：

| 考评方式 | 评语必填 | 结果                                                |
| -------- | -------- | --------------------------------------------------- |
| 整体     | 是       | 每个指标的评语、评分都必填<br>没有评分、等级        |
| 整体     | 否       | 每个指标的评语不必填、评分都必填<br>没有评分、等级  |
| 逐条     | 是       | 每个指标的评语、评分都必填<br/>没有评分、等级<br>   |
| 逐条     | 否       | 每个指标的评语不必填、评分都必填<br/>没有评分、等级 |



指标考评

| 分组允许员工添加指标 | 允许引用指标库 | 考评类型                                                     | 指标考评进度条         |
| -------------------- | -------------- | ------------------------------------------------------------ | ---------------------- |
| 是                   | 是             | 1、按考核考核<br>2、按分组考核（每个指标不能独立设置评估关系） | 展示分组的评定关系     |
| 是                   | 否             | 1、按考核考核<br/>2、按分组考核（每个指标不能独立设置评估关系） | 展示分组的评定关系     |
| 否                   | 是             | 1、按考核考核<br/>2、按分组考核（每个指标不能独立设置评估关系） | 展示分组的评定关系     |
| 否                   | 否             | 1、按考核表考核<br>2、按指标考核（每个指标可独立设置评估关系） | 展示每个指标的评定关系 |



总分=指标考核： 自评权重* 自评总分 + 考评权重 * （sum(目标分组权重 *  指标权重 * 指标分 * 难度系数) * 考评人权重【评估关系】）

考核表： 自评权重 * 自评总分 +  sum( 考评权重 * 考评人权重 * 综合评分)



指标分 = 指标分 * 难度系数 * 目标分组权重 * 自评权重 +   sum(指标分 * 难度系数 * 目标分组权重 * 考评权重 * 评估关系权重)

调整指标分 = sum( 指标分 * 权重)
