## 主存

```
主存：MAR(地址寄存器)、MDR(数据寄存器，用于临时存储数据)、存储体（实际存储数据的位置：按地址存储）
	存储单元：每一个存储单元存放一些二进制代码
	存储字：二进制代码的组合
	存储字长：每个存储单元的长度
	存储元：存储二进制的电子元件。每个存储元存储1bit
	
	
MAR的位数反应了存储体有多少个存储单元
MDR的位数反应了存储单元的位数 
如果MDR是16位的，1字就是16bit,1字节（B）固定等于8bit。所以一个存储单元是2字节
1B=8bit
```

## 执行流程

<img src="/Users/peilizhi/Library/Application Support/typora-user-images/image-20210624000813384.png" alt="image-20210624000813384" style="zoom:50%;" />

```
1、PC指向第一条指令的地址
2、地址总线读取MAR地址
3、MAR获取存储体中相应地址的数据
4、将数据临时放入MDR中
5、将MDR中的数据放入IR中
6、将操作码放入CU中，CU分析指令
7、读取指令中的操作数（执行3步）
8、将操作数放入运算器中相应的寄存器中
9、PC++
```

**指令包括指令码和操作数**

## 计算机性能

```
CPU主频：CPU内数字脉冲信号震荡的频率
CPU主频=1/CPU时钟周期
CPI：执行一条指令所需要的时钟周期数（一般取平均值。不同的CPU执行同一指令的时间不同）
（CPU执行时间）执行一条指令的耗时=CPI*时钟周期
IPS:每秒执行多少条指令
FLOPS：每秒执行的浮点数计算
数据通路带宽：数据总线一次并行传输的数据位数
吞吐量：系统在单位时间内完成处理的数量（系统的吞吐量主要取决于主存的存取周期）
```
