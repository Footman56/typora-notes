# 一、作用

```
存储数据，访问效率最高，支持随机访问。
一旦长度固定后就不能改变
```

数组用来存储**固定长度**的**同类型**元素。

# 二、定义

```java
int [] array=new int[5];  // 使用new关键字来定义
int [] array2={12,13,14}; // 直接创建并同时赋值，长度固定


如果是基本数据类型的话，定义好的数组内存在默认值。int:0,byte:0,double:0.0
如果是复杂对象的话，默认值就是null
```

# 三、方法

```
在使用List和Map的时候，可以直接通过size()方法获取容器的实际大小（长度）。
数组也有一个关键字length来获取大小（长度）,固定长度
```

```
Arrays 工具类
sort（）：对数组内数据排序，如果排序的元素不是String或者基本数据类型，就需要主动实现Comparable接口（基本数据类型和String本身已经实现了Comparable接口）。
swap（）： 根据下标交换数组内数据
binarySearch（）： 二分查找，必须是排序好的数组
asList()方法用来将数组转化成List（ArrayList）
toString()方法用来将数组进行字符串格式的输出
```
