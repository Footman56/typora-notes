   ```sh
   cd blog/huochai-blog
   ```

   ![image-20230129104753116](https://gcore.jsdelivr.net/gh/Footman56/imageBeds/202301291048007.png)

   ![image-20230129105217459](https://gcore.jsdelivr.net/gh/Footman56/imageBeds/202301291052491.png)

   

之后进入http://localhost:1313/ 地址查看实际效果

5. 推送到github

   ```shell
   # 将本次修改的文件添加到暂存区
   git add .
   # 提交到本地服务器
   git commit -m "日志"
   # 推送
   git push -u origin main 
   ```

   只有github有更新，都会重新部署，
