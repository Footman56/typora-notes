# 二、stack 主题添加评论

在config.yaml 中添加

```yaml
   waline:
            serverURL: https://waline.huochai.xyz
            lang: zh-CN
            visitor: false
            pageview: true
            comment: true
            reaction: true
            search: true
            dark: auto
            emoji:
                - https://cdn.jsdelivr.net/gh/walinejs/emojis/weibo
            requiredMeta:
                - name
                - email
                - url
            locale:
                admin: Kerm
                placeholder:
```
