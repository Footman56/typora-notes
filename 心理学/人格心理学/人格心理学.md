# 人格

是指在**遗传**与**环境**的交互作用下,个体形成的 典型的心理品质和行为倾向的特征

人格的特性

+ 独特性

  + 典型性：主要体现在个人核心人格的特征上。有特点
  + 差异性：没有两个人是完全一样的

+ 稳定性

  + 形成的稳定性：某种人格特点一旦形成，就相对 稳定，改变它是有一定难度的
  + 表达的稳定性：不同时间、情境下人格会保持稳 定，有跨情境稳定性和持续性

+ 统合性

  ​	个体的人格组成是多元复杂动态的人格元素群，不同人格元素按照一定**秩序与规则**有机结合起来。

+ 功能性

  ​	人格是个体生活轨迹的根源

  ​	人格**决定**一个人的生活方式,有时甚至会决定一个 人的命运

# 经典精神分析流派

弗洛伊德：人格理论的创始者

意识：能够觉察到的心理活动，是个体能觉知到的心理层次

前意识：介于意识和无意识之间的一种心理状态，是那些此时此刻虽然意识不到，但是在集中注意、认真回忆的情况下可以回忆起的经验

潜意识：人的本能冲动以及出生后被压抑的人的欲望。这些欲望往往因为社会行为规范不允许满足，而被压抑到内心深处，意识不能将其唤起。

## 三我模型：

+ 本我：最底层人格结构，人原始的无意识本能，动力成分【快乐原则】
+ 超我：人格最高层次，代表道德和伦理的一面【道德原则】
+ 自我；人格的中间层次，本我中分化出来，受现实制约的本能。 考虑现实性的基础下满足本我冲动，调节作用。【现实原则】

## 人格动力：

精神分析提出一个重要假设，即人的心理过程 需要**能量驱动**才能不断的进行下去，这种能力 就是心理能量。

+ 等量原则：心理的某部分占用了过多的能量，另一部分便得不到重组的能量
+ 平衡原则：当两个心理系统的能量不等时，能量会从较强的系统流向较弱的系统中

生本能：生本能表现为通过寻求食物、水、空气 等需要来**实现个体和群体的生存** 生本能指向生长 具有**积极、建设性**的作用

死本能：死本能指将**死亡的欲望**转向其他客体 而非自身的攻击性驱力 **“想死”**的念头 会使人们出现破坏、杀戮等**毁灭性行为**

## 人格适应

防御机制

不同的**防御机制内容不同**,但都具有**两个共同特征:**一是对**现实的否认或扭曲**;二是**在潜意识中运作**

+ 压抑：将意识中的令人不安的事物从意识层面压制回潜意识
+ 反向形成：表现出与令人不安的冲动的相反行为
+ 合理化：对引发焦虑的行为重新解释，使其更加合理，易被接受
+ 否认：拒绝承认外部威胁或者曾经发生过的创伤性事件的存在
+ 移情：通过转向其他对象来缓解焦虑的行为。有时情感的转移可以指向个体自身
+ 退行：个体因受挫折后的痛苦而退回到拥有更多快乐、没有挫折、焦虑的时期
+ 投射：将引发焦虑的事件转嫁到他人身上
+ 升华「唯一正能量」

## 特质

**奥尔波特**认为**特质**是决定个体行为的基本特 性,是构成人格的基本元素,也是评价人格的基 本单位。**人格**则是一个人的内在心理生理系统的动态 组织，是个体面对环境是独特的行为和思想

特质学说建立在两个假设之上

+ 人格特征在时间上相对稳定
+ 人格特征具有跨情境的稳定性



## 特质结构

奥尔波特

+ 共同特质：同一文化形态下 特 人们所共有、相 质 同的特质

  + 次要特质：不甚明显，一致性和概括性差的，只有在特殊场合下才表现出的

## 个人建构论

个人建构论基于这样的假设:没有客观的、绝对的真 理，一个现象是否具有意义，依赖于个体解释该现象 的方式。

建构是指个体解释和预测生活的认知结构。**建构是人 格的核心概念**，是个体用来解释世、分析他人观点， 对事件进行整理分类并控制事件的思维模式。如果没 有建构，人生就会一片混乱。 

人格差异主要是个体解释世界的不同方式导致的。



+ 核心建构

  核心建构是个体行为中最基本稳定的 建构，是决定个体行为一致性或同一 性的关键建构

+ 外围建构

  外围建构是比较容易改变的、不起决 定作用的建构，可以有一定的变化性



+ 可渗透建构

  可渗透性建构指能容许新成分 进入，有一定的可适用范围

+ 不可渗透建构

  非渗透性建构则指拒绝新成分 进入其领域的建构



+ 紧缩性建构

  紧缩建构是对时间的预测绝无改变的建构

+ 松散建构

  松散建构是对时间的预测可 随着时间、情境的不同而产生 变化的建构
