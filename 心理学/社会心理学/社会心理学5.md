# 社会促进

社会促进也称做社会助长，指个体完成某种任务时由于他人在场而提高了绩效的现象。他人在场的形式包括实际在场、想象在场和隐藏在场。

社会促进效应：

+ 结伴效应
+ 观众效应

优势反应强化说：

+ 个体熟练或简单机械性的，他人在场，效率更高

+ 个体正在学习的、不熟练的，需要非常复杂的过程的，他人在场，效率会更低

  

# 社会懈怠

磨洋工，就是一种社会懈怠现象，一般来说，个体在群体活动中，付出的努力水平都会下降，且群体规模越大，个人努力水平越低

也称社会逍遥，指群体一起完成一件任务时，个人所付出的努力比单独完成时偏少的现象

社会懈怠的原因：

+ 个体在群体活动中责任意识降低
+ 被评价的焦虑减弱

# 模仿

个体在没有外部压力的条件下，受他人的影响仿照他人，使自己与他人相同或相似的现象

与从众最大的区别就是 是否有外界压力

模仿的意义：

+ 模仿是学习基础

+ 适应作用

+ 促进群体形成

+ 模仿的发展：

  + 从无意到有意
  + 从游戏模仿到活动实践模仿
  + 从对外模仿到对内模仿

  塔尔德认为 模仿是基本社会现象，“一切事物不是发明，就是模仿”

模仿的原则

+ 下降率
+ 几何级数率
+ 先内后外率

# 暗示

暗示往往采用比较**含蓄、间接**的方式进行

暗示涉及三个要素，暗示者、暗示信息和被暗示者

非对抗的条件下，通过**语言、表情、姿势及动作**等对他人的心理与行为发生影响，使其
接暗示者的意见和观点，或者按所暗示的方式去活动

暗示分类：

+ 他人暗示和自我暗示
+ 有意暗示和无意暗示
+ 直接暗示和间接暗示
+ 暗示和反暗示

影响暗示的因素

+ 暗示者的**权力**、**威望**、**社会地位**以及**人格魅力**对暗示效果有明显影响
+ 被暗示者如果独立性差，缺乏自信，知识水平低，那么暗示效果就明显
+ 被暗示者的年龄，性别与暗示效果也有关系，年龄越小越容易被暗示，一般女性比男性容易接受暗示。
+ 被暗示者所处情境是暗示发生作用的客观环境个体处于困难情境且缺乏社会支持时，往往容易受暗示

# 社会感染

社会感染是一种**较大范围内**的信息与情绪的传递过程，即通过语言、表情、动作及其 他方式引起众人相同的情绪和行为

社会感染的特点：

+ 双向性
+ 爆发性
+ 接受的迅速性
+ 个体间的感染
+ 大型开发群体的感染
+ 大众传媒的感染

# 爱情

爱情是世界上最复杂的情感现象
**爱情是人际吸引最强烈的形式**。是身心成熟到一定程度的个体对异性个体产生的有浪漫
色彩的高级情感(狭义定义)

## 吊桥效应

**情绪=刺激**✖**生理唤醒**✖**认知标签**

印度学者 古普塔 ：由爱情结合的，**婚后5年**，彼此爱的情感开始不断减少 由父母之命结合的夫妻，开始爱情水平并不高，但他们的感情会慢慢增加，5年后 大大超过了因爱情而结合的夫妻



爱情与喜欢的区别

+ 依恋：卷入爱情的双方在感到 孤独时，会**高度特异性** 地去寻找对方来陪伴和 宽慰，喜欢的对象不会有这种效果
+ 利他：高度关怀对方的情感 状态，认为**对方快乐 和幸福是自己的责任**。 在对方有不足时，也 会表现出高度的宽容
+ 亲密：爱情中的双方不仅仅对 对方有高度的情感依赖， 而且会有**身体接触**的需 求。性是爱情的基础， 是爱情的核心成分之一

爱情发展阶段

1. 取样与评估
2. 互惠
3. 承诺
4. 制度化

爱情三角形理论：

斯腾伯格认为爱情由 **亲密**、**激情**、**承诺**组成

亲密：重视彼此的喜欢、理解与期待

激情：魅力与性的吸引

承诺：决定发展稳定的关系

# 婚姻

婚姻是男女结成夫妻关系的行为，是家庭成立的基础和标志,**婚姻关系的本质在于它 的社会性**。夫妻关系是一种特定的人际关系和社会关系

婚姻动机：

+ 经济
+ 繁衍
+ 爱情

夫妻关系类型

+ 爱情型
  + 由美貌、性吸引的结合
  + 人格的相似或者互补为基础，稳定
+ 功利型：以爱情之外的出身、学历、财产等为基础的结合。当夫妻双方收益与成败基本平衡时
+ 平等合作与分工型：平等合作型的夫妻双方平等分担家务，分工型的根据自身特 点进行分工，两者特点是均进入自己角色，对彼此有期待。 **彼此都能认识到双方在家庭中的价值**
+ 建设型：夫妻双方有共同目标，共同努力生活\工作， 围绕目标紧密合作，**达到目标后，一方可能满 足**，继而懒散，导致关系出现裂痕。 **共苦但不能同甘**
+ 惰性型：双方会**迅速失去**对婚姻的热情，不能发现问题，不 愿意进行新的尝试，**只希望按照老样子生活**
+ 失望型：现实不理想，理想不现实
+ 一体型：夫妻双方**把对方视作自己的一部分**，心 心相印。生活较为美满、稳定但是有些 封闭。

# 离婚

夫妻彼此心理的不协调、背离与对立、会造成双方的心理冲突
**心理冲突往往是离婚的原因和前奏**，而离婚往往是心理冲突激化的结果

定义：

+ 需求不满足
+ 价值观的不一致
+ 自我的远离
+ 夫妻的性差异

# 家庭

家庭作为一个群体，是社会的细胞，是社会生活的基本单位

家庭由婚姻关系、血缘关系及收养关系构成的

构成要素：家庭成员数量、代际层次、夫妻数量

+ 核心家庭：一对父母，多个未婚子女
+ 主干家庭：一对父母，一个结婚子女
+ 联合家庭：一对父母，多对结婚子女
+ 其他家庭


家庭的功能

+ 经济功能
+ 性功能
+ 生育功能
+ 抚养与赡养功能
+ 教育功能
+ 情感交流功能
+ 休闲与娱乐功能



影响家庭的因素

+ 社会与环境
+ 家庭成员素质
+ 家庭成员的人际距离
