# 社会角色

个体与其社会地位、身份相一致的行为方式以及相应的心理状态，他是特定地位的个体行为上的期待，**是社会群体得以形成的基础**

个体符合社会期待

## 社会角色划分

+ 按获得方式

  + 先赋角色：建立在先天因素基础上的角色（父母）
  + 成就角色：依靠个体努力获得的角色（教师）

+ 按角色功能

  + 功利性角色：追求实际利益为基本目标的角色（商人）
  + 表现型角色：表现社会秩序、制度、价值观念，道德风尚为目标的角色（学者）

+ 按角色行为规范话程度

  + 规定型角色：规范化程度高，个体自由度低（军人）
  + 开放型角色：规范化程度低，个体自由度高（朋友）

+ 按角色承担者心理状态

  + 自觉角色；对自己的角色扮演有较为明确的认知，并尽力感染“观众”
  + 不自觉角色：未意识到角色的扮演，只是习惯的方式行动（性别角色）

  

## 角色扮演

1. 角色期待：公众对其行为方式的要求和期望
2. 角色领悟：个体对角色的认识和理解
3. 角色实践：在前两者的基础上，个体在实际生活中，表现社会角色的过程

## 角色失调

+ 角色冲突：个体在不同条件下往往有者不同的地位、身份、角色，如果他们互不相融出现矛盾，个体在心里生感到角色冲突
  + 角色间冲突：同一主体内，两个或者两个以上的角色之间的矛盾所导致的冲突
  + 角色内冲突：人们对同一个角色不同期待所引起的冲突
+ 角色不清：个体对所扮演的角色人事不清楚，或公众对社会变迁期间出现的新角色人事不清，还未能形成对这一新角色的社会期待
+ 角色中断：由于各种原因使个体所扮演的角色发生中途间断的现象。比如从旧角色中退出来，却不知道或者来不及建立新的社会角色规范和行为准则
+ 角色失败：角色承担者不得不退出舞台，放弃原有角色【最严重的角色失调】

# 自我

自我是个体对自我存在状态的认知，包括自己的生理状体、心理状态、人际状态、社会角色的认知

## 自我的定义

【詹姆士】主我：认知的主体，是主动的自我，进行中的意识留动力成分

【詹姆士】客我：认识的对象包括一个人所持有的关于他自己的认识和信念，约束主我的活动

【米德】镜我：由他人的判断所反应的自我概念

罗杰斯：自我概念比真实自我对个体的行为以及人格又更为重要的作用，它是个体自我知觉的体系与认识自己的方式

## 自我的结构

+ 物质自我：自我的载体，基础，生理层面
+ 心理自我：态度、信念、价值观与人格特征的总合
+ 社会自我：个体扮演的社会角色，自我概念的核心
+ 理想自我：个体期待自己成为怎样的人，动力
+ 反思自我：自我概念的反馈

## 自我的功能

+ 保持个体内在的一致性
+ 解释经验
+ 决定期待

自我概念的形成与发展

1. 生理自我：8个月左右开始 3岁基本成型

   是自我概念的原始形态 个体对自己躯体的认知 包括占有感、支配感与爱护感等 其使个体认识到自己的存在

2. 社会自我：3岁到13、14岁

   人们能了解社会对自己的**期待** 并根据社会期待调整自己的行动

3. 心理自我：需要约10年 从青春期到成年

   个体能知觉和调节自己的心理活动及其特征和状态 **根据社会需要和资深发展的要求调控自己的心理与行为**

   表现出主动和独立的特点，强调自我价值与自我理想 发展了**自尊**和**自信**——自我概念中的两个主要成分



# 身份

是由个体的**社会地位**及**处境地位**决定的自我认同

社会地位决定的是地位身份，稳定，是身份的主体

 处境地位决定的是处境身份，易变

## 身份的特点

+ 客观性
+ 主观性
+ 多重性
+ 稳定性
+ 契约性

# 自尊

个体对其社会角色进行自我评价的结果

自尊公式：

自尊 = 成功/ 抱负

##  影响自尊的因素

+ 家庭中的亲子关系
+ 行为表现的反馈
+ 选择参与扬长避短
+ 根据相似性原理理性地进行社会比较

# 社会知觉

社会知觉包括个体对他人、群体以及自己的知觉

对他人、群体的是人际知觉；对自己的是自我知觉

普通心理学 ：知觉，指个体对直接作用于他的客观事物整体属性的反应是认知的初级阶段，不包括判断、推理等高级认知过程

社会心理学：包括复杂的认知过程，既有对人的外部特征和人格特征的知觉，又有对人际关系的知觉以及对行为原因的推理、判断与解释

## 影响社会知觉的主观因素

2. 认知者的动机与兴趣

## 印象

印象是个体头脑中有关认知客体的形象

个体接触新的社会情境时，一般会按照以往的经验将情境中的人或事进行归类，明确其对自己的意义使自己的行为获得明确定向。这一过程成为**印象形成**。

### 印象形成

+ 首因效应：最初获得的 信息的影响大于后来获得的信息
+ 近因效应：**最新**获得的信 息的影响大于原来 获得的信息

+ 光环效应：光环效应也称晕轮效应，指个体对认知对象的某些品质一但形成倾向性印象，就会带着这种倾向去评价认知对象的其他品质。

  是一种以偏概全的现象.

+ 刻板印象：人们通过自己的经验形成对某类人或某类事较为固定的看法叫做刻板印象。

  刻板印象具有社会适应的意义，能使人的社会知觉过程简化。但有限经验基础上形成的刻板印象往往有消极作用。

  偏见形成的一种因素

### 信息整合模式

+ 加法模式：形成总体印象时，参考的 是各种品质的评价分值总合
+ 平均模式：将各项特征的分值加以平 均，根据平均值的高低来 形成对他人的好或不好的 总体评价
+ 加权平均模式：用权重与每种特征的强度 相乘法，是从逻辑上判断 各种特征的重要性
+ 中心品质模式：根据重要的，对个体意义 大的特征来形成总体印象， 一般来说，**这是多数人日 常生活中采用的模式**

### 印象管理

印象管理也称印象整饰和印象控制

指个体以一定的方式去影响他人对自己的印象，通过一定的方法去影响别人对自己的印象形成过程，使他人对自己的印象符合自我的期待

印象管理的策略

+ 按社会常模管理自己
+ 隐藏自我与抬高自我
+ 按社会期望管理自己
+ 投其所好

# 归因

指个体根据有关信息、线索对自己和他人的行为原因，进行推测与判断的过程；不仅仅是一种心理过程

### 归因分类：

+ 内因、外因、综合归因

  + 内因：将行为原因归于个人特征
  + 外因：将行为原因归于外部条件
  + 综合归因：行为与事件的发生并非由内归因或外归因这样单一方面的因素引起，而兼有两者的影响

+ 稳定性原因、易变性原因

  + 稳定性原因：长时间内保持稳定的，如人格特征、能力等
  + 易变性原因：短时间会发生改变的，如气候条件，情绪等

+ 可控性原因 不可控原因

  + 可控性原因：

    个体可以通过主观努力改变行为及其后果，更可能对行为做出变化的预测

  + 不可控原因：

    表明努力也可能无力改变，较可能对未来的行为做出准确的 预测

### 控制点理论

【个体归因倾向性理论】：

由于内控者与外控者理解的控制点来源不同，因而他们对待事物的态度与行为方式也不相同

归因原则：

+ 不变性原则【海德】：找某一特定结果与特定原因之间的不变联系
+ 折扣原则：如果也存在其他看起来合理的原因，那么某一原因引起某一特定结果的作用就会打折扣
+ 协变原则【凯利】
  + 特异性：主体是否只针对客体反应
  + 共同性：不同主体是否针对客体反应
  + 一致性：行为主体在不同背景下，做出的反应是否相同

### 影响归因的因素

+ 社会视角：当事人和局外人对行动者的解释有差别
+ 自我价值保护倾向：
  + 自我访卫策略：我没有错
  + 动机归因偏差：别人成功是幸运；别人失败是能力不足
+ 观察位置：空间位置;凸显的更易受关注
+ 时间因素：时间推移导致归因具有情境性。人们倾 向于把**过去事解释为背景**的原因
