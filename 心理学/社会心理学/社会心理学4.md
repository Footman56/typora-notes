# 态度

态度是个体对特定对象总的评价和稳定性的反应倾向

## 态度的特点：

+ 内在性
+ 对象性
+ 稳定性

ABC模型

+ 情感成分
+ 行为倾向成分
+ 认知成分

三个成分不协调时，情感成分往往占有主导地位

## 态度功能：

+ 工具性：能带来利益
+ 自我防御功能：选择利于自我防御的态度
+ 价值变现功能：通过态度表现自己的价值
+ 认知功能：通过态度赋予意义

## 态度的维度

方向：个体对态度对象是肯定或否定指向

强度：倾向于某一特定方向的程度

深度：个体对特定态度对象的卷入水平，

向中度：某种态度在个体态度体系及相关价值体系中接近**核心价值**的程度

外显度：个体态度在其行为方向与行为方式上的外露程度

## 态度的形成：

1. 依从：态度形成的开始，是表面暂时的权宜之计，一种印象管理策略。
2. 认同：个体自愿接受他人观点信息等，使自己与他人一致。 此阶段情感因素起明显作用，依赖于对象对个体的吸引力
3. 内化：最后阶段。个体从内心相信并接受他人的观点，纳入自己的体系。 理性，即认知成分为基础

##  态度的转变

个体形成一定的态度后，由于接受新的信息或意见而发生变化。这个过程叫做态度转变

传递者-》沟通信息 -〉目标（接受者）〈-情境

传递者：新的信息提供者

沟通信息：新信息和旧信息的对比

目标：旧信息的持有者

情境：所处的大背景

态度转变影响因素

传递者

+ 传递者的威信
+ 传递者的立场
+ 说服的意图
+ 说服者的吸引力

沟通信息：

+ 信息差异

  威信高，差异大，引起态度转变较大

  威信低，差异适中，引发转变量大

+ 畏惧

  有利说服，但太强烈反而不好

+ 信息倾向性：

  一般公众，单一倾向更好；文化水平较高，提供正反面信息会更好

+ 信息提供方式：面对面更好

接受者：

+ 原有态度与信念的特性

  内化的，既定的，较难被说服

+ 人格因素

  依赖性强和高社会赞许动机更容易被说服

+ 个体心理倾向

  逆反心理，爱面子不容易被说服，利用自我防卫策略减少被说服

情境：

+ 预先警告

  不够坚定的，会被说服

+ 分心

+ 重复

海德认为：在人们的态度系统中，存在某个情感因素之间或者评价因素之间趋于一致的压力，如果出现不平衡，那么就会朝平衡转化

费力最小原则：使用最小的力量使平衡

P-O-X

P:个体，O-他人，X-另一个对象

P-O之间的关系最重要

P-O联系为肯定时的平衡为强平衡，不平衡时为强不平衡

P-O联系为否定时的平衡为弱平衡，不平衡为弱不平衡

平衡：相乘为正数

认知失调理论：

费斯汀格：个体关于自我、环境、态度对象都有很多认知因素，当各认知因素出现 非配合性的关系时，个体就会产生失调

引起认知失调的原因：

+ 逻辑的矛盾
+ 文化价值的冲突
+ 观念的矛盾
+ 新旧知识经验相悖

减少认知失调

+ 改变或者否定认知失调的一方
+ 引入或者新增新的认知因素
+ 降低失调认知因素各方的强度

社会交换论从个体对得失权衡比较后，产生的趋向与回避动机的角度，来解释态度的形成与转变。

# 沟通

沟通指信息的传递态和度交转流变的就过是程说，服包的括过人程际沟通和大众沟通

基本结构

+ 传递者
+ 信息
+ 渠道
+ 接受者
+ 反馈
+ 障碍

沟通的主要作用：

+ 是满足需求、维持心理平衡的重要因素
+ 是获取信息的手段
+ 是思想交流与情感分析的工具
+ 是减少冲突，改善人际关系的重要途径
+ **能协调群体活动**,促进效率的提高与组织目标的实现

沟通分类：

+ 正式与非正式沟通
+ 上行沟通、下行沟通与平行沟通 
+  单向沟通与双向沟通
+ 口头沟通与书面沟通
+ 现实沟通与虚拟沟通

在正式群体中，成员之间信息的交流与传递的结构成为正式沟通网络

+ 链式
+ 轮式
+ 圆周式
+ 全通道式
+ Y式

学者通过对“小道消息”的研究，发现非正式沟通网络主要有三种形式

+ 流言式
+ 集束式
+ 偶然式

## 身体语言沟通

身体语言是非语词性的身体符号，包括目光与面部表情，身体运动与触摸，姿个体形成一定的态度后，由于接受新的信息或意见而发生变化。这个过程叫做势与装饰，人际距离等。人们可以通过身体语言表达丰富的意义。专门研究身体语态度转变。言的学科是身体语言学

非语词性的身体符号

+ 眼光与面部表情

  人类 对**目光很难做到随意控制**。 **眼睛是内心情感的灵敏指示器**。目光接触是非常重要的身体语言沟通方式。

  面部表情表达的感情有可能与实际的情况不一致。

+ 身体运动与接触

  身体运动是最易为人发现的一种身体语言。手势语占有重要位置

  人际沟通的有力方式，个体在与他人触摸和身体接触时的情感体验最为深刻

+ 姿势与装饰

  姿势是个体运用身体或姿态表达情感及态度的身体语言
  服装、化妆、饰品和携带物也能透露一个人的情趣、爱好、情感、态度等

+ 人际距离

  人际距离是沟通与交往时，个体身体之间的空间距离

  + 公众距离（12～25英尺）
  + 个人距离（1.5～4英尺）
  + 社交距离（4～12英尺）
  + 亲密距离（0～18英寸）

# 人际关系

人际关系是人与人在沟态通度与转交变往就中是建说立服起的来过的程,直接的**心理上**的联系

人际关系的特点：

+ 个体性：个人的主要爱好
+ 直接性：个体能直接感受到存在
+ 情感性：人际关系的基础是情感联系

人际关系的发展阶段

1. 定向阶段
2. 情感探索阶段
3. 情感交流阶段
4. 稳定交往阶段

## 自我暴露：

自我暴露也称自我开放，指在沟通和交往的时候把自己私人性的方面显示给他人。

自我暴露的广度和深度是人际关系深度的一个敏感的“探测器”。

1. 情感爱好
2. 态度
3. 自我概念、人际关系
4. 隐私

良好人际关系原则：

+ 相互性原则
+ 交互性原则
+ 自我保护价值原则
+ 平等原则

人的三种基本人际关系需要：

+ 包容需要
+ 支配需要
+ 感情需要

破裂的时候是反方向破裂

## 从众

从众是指在**群体压力下**，**个体**在认知，判断，信念与行为等方面**自愿**的与多数人保持一致的现象。俗称随大流。社会心理学研究较多的是行为方面的从众。**自愿但被动**

特点

1. 压力是真实存在的
2. 群体压力可以在个体**意识到**的情况下 发生作用，使个体通过**理性抉择**，选择从众; 也可以在**没有意识到**的情况下发生影响，使 人**不自觉**地跟随多数人行动。
3. 从众行为有时候虽然**不符 合个体的本意**，但却是个体的**自 愿行为**。

类型：

+ 真从众：表里一致的从众，行为与认知不存在冲突
+ 权宜从众：行为与群体一致，认知不一致
+ 反从众：内心与群体一致，行为却相反（律师）

自愿是从众的重要特点

从众原因：

+ 寻找行动参照
+ 对偏离的恐惧
+ 群体凝聚力

影响从众因素：

+ 群体因素
  + 一致性
  + 凝聚力
  + 群体规模
+ 个体人格因素
  + 自我评价：自我评价高，不容易从众
  + 独立性：独立性高，不容易从众
+ 情境的明确性
  + 情境明确，从众行为会减少
+ 其他因素
  + 性别
  + 智力
