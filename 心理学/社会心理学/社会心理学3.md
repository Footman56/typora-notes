# 社会动机

动机是引起、推动、维持与调节个体行为，使之趋向于一定目标的心理过程或者内在动力

由人的自然属性，自然需要引起的动机称为自然动机；有人的社会属性、社会需要引起的动机称为社会动机

社会动机是人的社会行为的直接原因

## 功能

+ 激活功能：激发个体产生社会功能，使个体处于活动状态，是行为的启动因素
+ 指向功能：社会行为总是指向一定目标，动机使行为具有明确的指向性和目的性
+ 维持和调节功能：个体的社会行为达到目标前，社会动机起维持作用。若行为受阻，只要动机存在，行为就不会完全停止，将以别的形式继续存在。

## 亲和动机

个体害怕孤独、希望与他人在一起，建立协作友好联系的一种心理倾向。亲和就是合群，是人际吸引较低的层次

**亲和动机的起源就是依恋**

亲和动机的作用

+ 满足个体社会性需要
+ 获得信息
+ 减轻心理压力
+ 避免尴尬

影响亲和动机的因素

+ 情景因素：压力越大，群体成员亲和动机越强，
+ 情绪因素：恐惧情绪越强烈，亲和动机越明显，高焦虑者的亲和倾向较低
+ 出生顺序：长子、长女恐惧时的合群倾向，要比弟妹更明显

## 成就动机

个体在各种情景下，追求成功成就的动机

成就动机的重要性

+ 个体层面：高成就动机使个体更冒险
+ 社会层面：个体都能有较高的动机水平，社会更加发展

抱负水平：

个体抱负水平越高，成就动机也就越高

 成功经验可提高个体的抱负水平

影响成就动机的影响因素：

+ 吸引力越大，成就动机越强
+ 机会越多，成就动机越强
+ 风险与成败的主观概率

## 权利动机

个体希望影响和控制他人的心理倾向

积极的权利动机：竭力去谋求领导职位

消极的权利动机：害怕失去权利

影响因素：

+ 社会控制的需求
+ 无能的恐惧

## 侵犯动机

侵犯行为也叫攻击行为，暴力行为，是个体有意伤害他人的行为侵犯，是由内部侵犯动机引起的

个体有意伤害他人，以使自己获取平衡和满足的一种心理倾向

### 侵犯的社会构造

侵犯是由于内部侵犯动机引起的：包括伤害行为、侵犯动机、社会评价

### 侵犯的原因

【弗洛伊德】本能论：死本能与生本能相互对立，只要活着，死本能受到求生欲望的妨碍，因而对内的破坏倾向转向外部，以侵犯的形式表现出来

【罗伦兹】：侵犯是一种本能，具有生物保护意义。动物通过侵犯保护其求食，生存的领地使物种能传承。同类之间的侵犯不一定以毁灭为结局，可能是以失败者的让步为目的。

挫折-侵犯学说：挫折即指阻碍个体达到目标的情景，也指行为受阻时，个体产生的**心理紧张状态。**

【多拉德】：**侵犯永远是挫折的一种后果**

+ 侵犯强度同目标受阻强度呈正比
+ 抑制侵犯的力量与该侵犯可能受到的预期惩罚强度呈正比
+ 若挫折强度一定，预期惩罚越大，侵犯发生可能性越小

【米勒】：挫折也可以导致侵犯以外的结果

【伯克威滋】：挫折导致的是侵犯的**情绪准备状态**，即愤怒而不是侵犯本身

社会学习论：侵犯行为是学习的，学习机制:联想，强化，模仿

影响侵犯动机

+ 情绪唤起水平
+ 道德发展水平
+ 自我控制水平
+ 大众角色与群体
+ 大众传媒的影响

## 利他动机

利他行为是利他动机支配的行为，是个体有益于他人、公众和社会，不期待回报的行为

个体不顾自身，增进他人的价值和利益的一种心理倾向

社会生物学观点：利他行为并非人类独有，动物也有利他行为。利他是动物以个体的“自我牺牲”换取物种存在和 延续的一种本能。

社会规范论：交互性规范与社会责任规范是利他行为的主要原因

影响利他动机的因素：

+ 自然环境
+ 社会环境
  + 他人在场
  + 情境的社会意义
+ 时间压力
+ 利他对象的特点

利他者的心理特征

+ 心境
+ 内疚
+ 人格

**利他技能**:救助技能与手段的掌握，会增加人们利他行为发生的可能性

# 社交情绪

个体一种主观体验。是个体的社会需要是否获得满足的反映

+ 社交焦虑：一种与人交往的时候，觉得不舒服、不自然、紧张甚至恐惧的情绪 体验。往往伴随生理上的症状。是一种消极的情绪体验。离开后会 回放
+ 嫉妒：1.针对性 2.持续性 3.对抗性 4.普遍性
+ 羞耻：因为自己在人格、能力、外貌等方面的缺憾，或者在思想和行为方面与社会常态不一样，产生的一种痛苦的情绪体验，过多过少都不利
+ 内疚：个体认为自己对实际或想象的罪行或过失负有责任，而产生强烈的 不安、羞愧和负罪的情绪体验。健康的内疚感是心灵的“报警器

特异高，共同性高，一致性高，归因于刺激客体
