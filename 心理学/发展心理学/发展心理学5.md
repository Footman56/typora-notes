# 青年期

## 青年期一般特征

+ 生理发育和心理发育达到成熟水平
+ 进入成人社会，享受社会权利、承 担社会义务
+ 生活空间扩大
+ 开始恋爱、结婚

## 青年期的思维发展

1. 二元阶段：非黑即白
2. 相对性阶段：通过对知识和真理 相对性的认知，并 通过比较来审视不 同的观点
3. 约定性阶段：既能坚持用约定俗成的立场和观点来认识问题，又能具体分析，从不同的立场调整认识

## 青年期的个性和社会性的发展

## 自我概念的发展

自我概念的特点：

1. 自我概念的抽象性日益增强
2. 自我概念更具组织性和整合性
3. 自我概念的结构更加分化

埃里克森认为青年期自我同一感的确立是 自我分化与整合统一的过程

+ 社会也给予青年暂缓履行成人的责任和义务的机会

人生观-- 人们对人生目的和意义的根本看法和态度



科尔伯特：道德推理发展理论  三个水平、六个阶段

+ 第一水平【前习俗水平 享乐】
  + 第一阶段:惩罚和服 从取向 (服从权威， 避免受罚，行为后果 作为是非标准)
  + 第二阶段:功利取向 (满足个人需要)
+ 第二水平【习俗水平  权威】
  + 第三阶段:”好孩子”取向(取悦别人就是好的)
  + 第四阶段:“好公民”取向(维护社会秩序，尽职尽责就是好的)
+ 第三水平【后习俗水平 良心】
  + 第五阶段:社会契约取向 (社会价值和个人权利作为是非标准)
  + 第六阶段:普遍道德原则 取向(是非是一种个人依 照普遍原则所确立的哲学)

科尔伯格 道德认知发展理论的要点

+ 道德发展具有固定不变的顺序
+ 达到后习俗水平的个人不多
+ 环境和社会文化因素只能决定道德发展的内 容和速度，不能影响道德发展顺序

# 老年期

## 老化的原因

1. 遗传学说:精神机能的老化、行为的变化以及随年龄增长而出 现的心理变化， 都是由遗传决定的，衰老是按遗传程序实现的 

2. 行为老化学说:由于精神退化机能引起的，并主要从行为变化 中表现出来

3. 强调个体和社会相互作用 
4. 疏离学说:老年人与社会的脱离是造成个体老化的主要原因 
5. 适应学说:老年期的主要变化是人际关系的改变



老年丧失期观点：老年期的心理变化只有衰退，没 有发展，是人生获得的丧失期

丧失的内容:身心健康、经济基础、社会角色、生活价值；心理随年龄的增长而发展，到 老年期，随年龄增长而衰退是 个体心理发展的总趋势

毕生发展观：任何时候都在发展中；**知觉发展最早、衰退也最早** 发展过程中既有发展也有衰退 ；**成熟因素影响儿童期、社会文化因素影响成熟期、个人因素对老年期影响最大**

## 老年期认知变化

感知觉衰退最早，变化最明显

视觉衰退-》听觉衰退-〉味嗅视 觉迟钝

老年期记忆衰退的特点

+ 老年机械记忆衰退明显；意义记忆相对衰退慢
+ 再认能力逐 渐老化，但  再认比回忆 较难保持较好
+ 识记和回忆 姓氏比较难

老年期主要的记忆障碍

+ 老年记忆障碍主要在于信息提取困难
+ 老年人记忆障碍是**编码储存**和**提取**过程相互作用结果
+ 老年人较少主动地运用记忆策略和方法
+ 文化因素对记忆的影响显著



对老年期记忆衰退的解释

+ 加工速度理论（速度减慢）：加工速度减慢是老年人认知 (记忆)减退的主要原因
+ 工作记忆理论（容量变小）：老年人发生认知(记忆)功能 衰退是因为他们缺乏信息加工 资源；工作记忆容量随年龄增长而变 小是老年期记忆减退的另一根 本原因

## 老年期的人格特征

老年人的人格表现出基本稳定的倾向。这说明老年人人格的基本类型和基 本特征不容易发生大的变化

老年期人格特征变化

+ 不安全感
+ 孤独感
+ 适应性差
+ 拘泥刻板，趋向保守
+ 回忆往事

原因：生物学因素、心理上的变化、社会文化因素
