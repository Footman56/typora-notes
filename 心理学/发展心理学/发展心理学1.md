# 什么是发展心理学

发展心理学是**研究心理**的发生、发展规律的科学，属于基础学科

**明确而稳定的条件反射的出现 - 动物心理学开始的标志**

# 特性

+ 整体性：不是相加、有机联系，是相互作用
+ 社会性：个体心理发展受社会条件制约
+ 活动性：动作和活动是主客观相互作用的中介
+ 规律性：
  + 普遍性、特殊性
  + 方向性、顺序性
  + 不平衡性
    + 第一发展加速器（出生～3，4岁）
    + 第二发展加速器（青春期）

# 研究内容

+ 年龄阶段特征

  1. 年龄特征 是本质特征

     心理发展与年龄由密切联系，但绝非同步

  2. 年龄阶段的根源

     不能年龄阶段的个体对来自客体的输入信息的加工过程是不同的

+ 心理发展的基础理论

  + 心理发展的动因【本质问题】：遗传和环境的作用

  + 连续性和阶段性的问题【过程问题】 

    1～3 语言发展、3～7岁人格发展 青年期结束进入成熟期

  + 内动力和外动力的关系（自生成vs环境教育）

  + “关键期‘问题

# 研究方法

## 功能：

+ 描述：描述研究对象的心理特点和发展状态
+ 解释：解释对心理发展现象以及变化的原因
+ 预测：通过观察现有发展规律推测研究对应的未来发展
+ 控制：对环境变量加以严格调控，以保证研究的预期
+ 特殊性： 
  + 发展性：心理发展的过程性和动态性（研究的核心特点）
  + 年龄跨度大（研究的显著特征）

横向研究： 相同的时间，不同年龄组，效率高；人为因素、阶层影响

纵向研究：相同的研究对象，不同时间；以概括连续性和发展规律；效率差，易流失，练习和疲劳效应

纵横交叉：先横后纵

高能心理机能：

+ 随意化
+ 概括-抽象
+ 整体化
+ 个性化

教育和教学于心理发展的关系：

+ 最近发展区
+ 教学先行
+ 最佳期限

## 主体客体 相互作用论

皮亚杰

一、儿童心理发展是主体客体相互作用的结果

1. 相互联系、相互制约，相互依存，缺一不可
2. 相互转化的互动关系，先天遗传可被环境改变
3. 相互作用受个体主观能动性的调节

二、认知发展本质的适应理论和主动构建学说

智慧就是**适应**的，最高形式的适应

旧图式（思维）- 》 同化 -〉 顺应 -》 平衡

同化：主体将环境中的信息纳入并整合到已有图式过程，去加强和丰富主体的动作

顺应：当主体的图式不能 适应客体的要求时， 发展的趋势 改变原有图式或创 造新的图式，以适 应环境需要

平衡：主体发展的心理动力，主动发展趋势

三、**心理起源于动作，动作是心理发展的源泉，是主客体相互作用的中介**

四、影响儿童发展心理的的基本因素

+ 成熟
+ 经验
+ 社会环境
+ 平衡化：同化和顺应

## 认知发展论

+ 感知阶段（0～2岁）

  凭借感知和动作经验适应外部环境，大约在9-12个月， 获得客体永久性

+ 前运算阶段（ 2～6、7岁）

  凭借心理符号 (主要是表象)进行思维， 但不能很好地掌握概念的概括性和一般性

  + 泛灵论（一切皆有生命）
  + 自我中心（在自己角度看问题）
  + 不能理顺整体和部分的关系
  + 思维不可逆
  + 缺乏守恒

+ 具体运算阶段 （6、7～11、12）

  + 具有守恒性
  + 脱离自我中心性（8，9岁是转折点）
  + 可逆性

  需要具体事物的支持

+ 形式运算阶段（11、12岁以后）

  + 思维发展到抽象逻辑推理水平
  + 思维形式摆脱思维内容
  + 进行假设—演绎推理

## 人格发展论

也称“心理社会发展阶段理论”，是毕生发展理论

人的心理危机是个人需要与社会要求不相适应至失调

社会环境决定心理危机能否得到积极解决

特点：

1. 八个阶段，序列发展
2. 每个阶段都有特点的发展任务
3. 每个阶段都有心理社会危机
4. 自我功能的发展居主导地位



+ 婴儿期（0～2岁）

  信任感-怀疑感 品质：希望，安全感

+ 婴儿后期（2～4岁）

  自主感-羞耻感 品质：意志 ，有意识控制大小便

+ 幼儿期（4～7岁） 主动感-内疚感 品质：目标
+ 童年期（7～12岁） 勤奋感-自卑感 品质：能力
+ 青少年期（12～18岁） 角色同一感-角色混乱感 ，品质：诚实
+ 成年早期（18～25岁）亲密感-孤独感 品质：爱
+ 成年中期（25～50） 繁衍感-停滞感 品质：关心
+ 成年晚期（50岁以后） 完善感——失望感 厌恶感 品质：智慧、贤明



内动力：人类所有的自生成、自发展的心理动力

外动力：家庭和社会文化环境对儿童的影响和教育作用

儿童心理发展是内动力和外动力相互作用的结果



# 儿童心理发展的关键期

关键期:儿童在某个时期最容易习得某种知识和 技能，或形成某种心理特征， 而过了这个时期 有关方面的发展会出现障碍，人类只在感受系统 范围内有所表现(不宜普遍适用)

并非完全不可弥补，故采用“敏感期”较为恰当

# 简史

儿童学研究基础：达尔文（一个婴儿的转略）

德国普莱尔 《儿童心理》：科学儿童心理学诞生的标志

儿童心理学到发展心理学的演变:1957年，美国《心理学年鉴》用“发展心理学”取代“儿童心理学”这一章的标题



# 错题

埃里克森认为幼儿期良好的人格特征是目标品质

成熟是训练的前提条件

**皮亚杰 认为影响儿童心理发展基本因素包括：成熟、经验、社会环境、平衡化**

埃里克森划分人格发展阶段的标准是 **心理社会危机**

学习的最佳期限的条件是 心理技能开始形成而尚未成熟之时

维果茨基的心理发展观认为：人的高级心理机能是由社会文化历史因素决定的

最近发展区是指：在有指导的情境下，儿童借助成人帮助所达到解决问题的水平与独立活动中所达到的解决问题水平之间的差距

埃里克森 认为幼儿期主要的发展任务是 获得主动感、克服内疚感

发展心理学研究的两个重点内容是 心理发展的年龄阶段特征、心理发展的基本理论

二因素论 ：心理发展是由遗传和环境两个因素决定的、企图揭示各因素单独发挥作用的程度、遗传和环境是两个各自独立的因素

维果茨基认为 低级心理机能转化到高级心理机能的标准是：心理活动的随意机能的形成和发展、心理机能相互作用并重新组合、心理活动抽象概括机能的形成和发展、心理活动越发突显个性特征

学习的最佳期限的前提和条件是：以个体发育成熟为前提、要以一定的心理技能发展为条件

维果茨基 提出教育、教学与心理发展的几个重要问题：最近发展区思想、教学应当在发展前面、学习和指导的最佳期限
