# 新生儿的发展

人的生命从受精卵开始，经历280天 40周的胎内发展，称作产前期

新生儿是指从出生到1月 的婴儿

+ 胚种期：0～2周
+ 胚胎期：3～8周
+ 胎儿期：9～38周

##  新生儿的条件反射

+ 具有明显生存意义的无条件反射

  + 食物反射
  + 防御反射

+ 没有明显生存意义的无条件反射

  4-6月自行消退，抓握反射

+ 定向反射

新生儿无条件反射的意义：

1. 有利于新生儿适应环境与保护自身

2. 发展早期的社会关系

3. 是各种条件反射建立的基础

   **无条件反射 是智力发展的最原始的基础**

## 生活行为模式

睡眠-觉醒周期模式

+ 啼哭 与看护人交流，
+ 睡眠  睡眠不规律可能是异常问题征兆
+ 觉醒时的活动 【每天2·3】

## 心理发生

+ 以感觉的产生为指标
+ 以无条件反射为指标
+ 以出现明确而稳定条件反射为指标

# 婴儿生理和动作的发展

婴儿的大脑 具有可塑性、可修复性

可塑性：婴儿大脑的大小和功能都受后天经验影响和制约

可修复性：婴儿早期大脑具有良好修复性

​	5 岁前语言中枢受损伤， 另一侧脑半球很快会产生替代性功能，使语言中枢转移(可修复)

​	0-2岁50%，4岁两倍，11岁修剪



动作发展对婴儿心理发展的意义

+ **动作是 婴儿心理发展的源泉**
+ 婴儿心理发展水平的指标
+ 使婴儿获得探索环境的新手段和主动权
+ 促进婴儿认知和社会交往能力发展

### 婴儿动作发展

+ 抓握技能

  发展要点：五指分化、手眼协调

+ 独立行走 婴儿发展的重要里程碑

发展原则和顺序

+ 头尾原则（从上到下）
+ 近远原则（由内向外）
+ 大小原则（大动作向精细动作）

# 婴儿学习

婴儿学习活动最早出现 在**胎儿末期**

模仿学习

条件反射学习方式（最基本的学习方式）

偏好新颖刺激的学习方式 ；习惯-去习惯化

# 婴儿的认知发展

感知觉是个体认知发展中最早出现，也是最先成熟的心理过程，

**感知觉是婴儿认知的开端**

## 婴儿感觉的发展

1. 视觉技能的发展

   视觉集中、追踪(2个月以 后)、颜色视觉、对光察觉和视觉敏锐度

2. 听觉技能的发展

   辨别能力、语音感知、音乐感知、视听协调

婴儿更喜欢复杂的形状、更喜欢面孔盘

## 婴儿知觉的发展

1. 跨感觉通道的知觉

   最明显的表现形式是手眼协调(精细动作)和视听协调(听音辨位)

2. 模式知觉 

   由部分组织成整体的能力

3. 深度知觉

   **吉布森**运用“**视觉悬崖** 装置”研究婴儿的深度知觉。结果发现，**约从6个 多月开始，婴儿就具有深度知觉**

## 婴儿记忆的发展

### 婴儿的注意

婴儿的注意最早表现是先天的定向反射

### 婴儿的记忆

按记忆时间划分：个体记忆发生的时间是**胎儿末期**

按记忆内容划分：12月以前是情绪记忆+ 动作记忆；12月以后是表象记忆+词语记忆

## 婴儿的加工整合信息能力与问题解决能力的发展

加工整合信息能力：9-12 个月婴儿能将食品、动物、交通工具等分别归类

问题解决能力： 8-11 个月婴儿的问题解决过程经历三个水平

+ 第一阶段：无效尝试
+ 第二阶段：有效尝试
+ 第三阶段：无须尝试就直接成功

# 婴儿言语的发展

## 婴儿发音的阶段性

+ 简单发音阶段：0～3个月，

  单音节发音阶段，婴儿以发出基本韵母为主很少有声母

+ 连续音节阶段： 4～8个月

  多音节发音阶段 婴儿能够连续重复地发音 发音活跃期

+ 学话萌芽阶段  9～12个月

  婴儿能将不同音节连续发出 出现声调起伏变化 像在说话但无明确意义指向

## 婴儿发音的特点

第一，不同民族和国家的婴儿最初发音呈现出普遍规律性

第二，婴儿真正掌握母语的各种发音，要到第一批词出现时才 能开始

第三，3 岁左右婴儿基本上能掌握母语的全部发音

第四，1.5 岁-2.5 岁是婴儿掌握母语基本语法的关键期

## 婴儿词汇的发展

### 词汇量的发展

婴儿在 **1~1.5 岁之间掌握第一批词汇**，其数量在 50 至 60 个左右。**3 岁词汇量增加到 1000 个左右**

### 掌握词汇的特点

1. 婴儿的词汇是从所熟悉的事物的名称开始
2. 婴儿理解的词义与成人不尽相同，或扩大、缩 小词义，或部分与成人的理解重叠

## 婴儿句子发展

单词句（1～1.5岁） -》 多词句 （1.5岁-3岁）

简单句 -〉 复合句 （2岁开始）

## 婴儿与成人之间的言语交往

**婴儿与成人前言语交往--手势**

**婴儿与成人言语交往--自己所知道的东西**

**成人与婴儿的言语交往-**

+ 句子要简短
+ 语速要减慢
+ 话语多重复
+ 副言语应用

**1.5—2.5 岁的婴儿是掌握母语基本语法的关键期，3 岁末可基本掌握母语语法规则系统**

# 婴儿个性和社会性的发展

## 婴儿气质

### 按活动特性划分

巴斯

+ 情绪性（负面情绪占优势）
+ 活动性（较多攻击行为）
+ 冲动性（情绪反应强烈）
+ 社交性（自来熟）

### 按综合维度划分

托马斯、切斯

+ 容易抚养型

  生活有规律，节奏明显;容易适应新环境;积极主动探索;愉 **抚养型** 快情绪多;情绪反应适中(占比40% )

+ 抚养困难型

  节律、生活规律性差;难应适应新环境;缺乏主动 探索的积极性;负面情绪多;情绪反应强烈(占比10% )

+ 发展缓慢型

  环境适应慢;对新事物反应消极;情绪经常不甚愉快;心 境不开朗(占比15% )

+ 混合型

  可能有相对快乐的心境，但面对新情境却有消极反应(占比35% )

特征：

+ 婴儿气质的稳定性

  在出生后第一年，婴儿气质的稳定性呈连续增长的模式

  **气质的稳定性是中等程度的稳定性**

+ 婴儿气质的可控性和可变性

  **婴儿的气质在它与环境的相互作用中是可以控制和改变的**

  1. 婴儿气质对早期教育的影响--不同气质类型的婴儿对早期教育的适应性和要求均不相同

  2. 早期教育对婴儿气质的影响--关键在于环境教育的要求与婴儿的气质特性是否相符合和适应

## 婴儿基本情绪的发展

最初的情绪都是生理性的、本能的反应

婴儿早期情绪是生物-社会现象

### 婴儿兴趣的发展

+ 先天反射反应阶段 （出生到百日前后）
+ 相似性事物再认知觉阶段（半岁前后）
+ 新异性事物探索阶段（1岁前后）

### 婴儿的社会性微笑

+ 自发性微笑（0～5周），不具有社会性
+ 无选择的社会性微笑（5周～4个月） 对任何人都微笑
+ 有选择的社会性微笑（4月以后） 只对熟悉的人微笑

### 婴儿的社会性哭泣

+ 自发性哭
+ 应答性的哭
+ 主动操作性的哭

### 分离焦虑

分离焦虑与陌生人焦虑是指婴儿在离开母亲,遭遇陌生人和陌生环境的情况下，产生惊恐、躲避 反应

1. 最初阶段：拒绝陌生人
2. 第二阶段：情感冷漠
3. 第三阶段：寻找可亲近的陌生人

### 情绪对生存和发展的意义

1. 情绪是婴儿早期适应环境的首要心理承担者
2. 情绪是激活婴儿心理活动和行为的驱动力
3. 情绪的社会性参照功能(符号功能)

## 婴儿的社会性依恋

依恋是婴儿与主要抚养者**(通常是母亲)**之间的最初的社会性联结， 也是婴儿情感社会化的重要标志

1. 第一阶段【社会反应阶段】：无差别的-人人如此
2. 第二阶段【社会反应阶段】：有差别的-偏爱熟人
3. 第三阶段【联络阶段】：特殊的情感-恋母情结
4. 第四阶段 互惠关系形成阶段

婴儿的依赖类型：

+ 安全型依恋
+ 回避型依恋
+ 反抗型依赖

早期教养对依恋的影响

1. 早期社会性依恋对日后人格特征的影响。安 全依恋型的孩子在成人后具有高自尊
2. 早期依恋类型影响个体内在工作模式的形成

衡量婴儿母亲教育方式的三个标准

+ 反应性
+ 情绪性
+ 社会性刺激

## 婴儿自我的 发展

哈特提出：客体我自我意识的出现是个体自我意识发展的第一次飞跃

主体我的自我意识（一周岁前后）

+ 婴儿把自己作为活动主体的认知
+ 婴儿能把自己与他人分开

客体我的自我意识（两周岁前后）

+ 婴儿开始把自己作为客体来认知
+ 能运用人称代词“你、我、他”称呼自己和他人， 如用“我”表示自己

人格发展论-埃里克森
