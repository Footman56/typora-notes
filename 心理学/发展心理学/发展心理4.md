学习是小学儿童的主导活动

教和学是师生双向互动的过程

小学儿童的学习逐渐转向掌握间接经验为主

**“学会学习”是小学生最基本的学习任务**

学习促进小学儿童心理积极发展



# 童年期的认知发展

## 记忆发展

小学阶段额儿童记忆策略

+ 复诵策略：
  + 有意识的重复、 诵读、诵习
  + 大部分儿童能自 发运用复诵策略 记忆
  + 容易接受记忆策 略训练
+ 组织策略
  + 组织策略:根据 记忆内容的意义 联系归类成系统 帮助记忆的策略
  +  归类识记可按概 念、 功用、颜色、 图形等标准组织 材料
+ 系统化策略
  + 对记忆材料进行信 息加工，将相互有 关联的信息按体系 关系进行整理并条 理化，组成知识系 统以帮助记忆
+ 巧妙加工策略
  + 有意识记忆超过 无意识记成为主 要
  + **有意义记忆逐渐 占主导地位**
  + 抽象记忆的发展 速度逐渐超过形 象记忆

## 思维发展

童年本质特征： **依赖具体内容的逻辑思维**

思维发展的质变过程：由具体形象思维向以抽象逻辑思维过度，思维的主导类型发展质变

**9～10岁 小学中年级阶段，是思维类型的转折年龄**

概括能力的发展

+ 概括取决于事物的外表【直接形象】
+ 能够进行形象抽象的概括【事物属性】
+ 以事物的本质特征和内在联系为主，初步接近科学概括【理解内涵】

推理能力发发展

+ 演绎推理：由多个判断推出新的判断

+ 归纳推理：部分对象具有某种性质推出所有对象由有具体某种性质

+ 类比推理：根据两个对象的一定关系，推理出其他也具有这种关系的两个事物

  小学儿童类比推理能力发展低于演绎推理和归纳推理

童年期新的思维结构形成

掌握守恒 -》 形成守恒概念的推理方式

+ 6～8 ：数概念守恒和长度守恒
+ 7～9 液体守恒和物质守恒
+ 8、9、10 **面积守恒和重量守恒**
+ 11～12 容积守恒

恒等性、可逆推理、两维互补推理

自我中心表现和脱自我中心化

+ 4-7岁 儿童具有自我中心现象，但并不是认知的主要成分

+ 8岁左右儿童处于脱自我中心的转折时代

+ 9岁以后 正确认知结果占主导地位，并基本上摆脱了自我中心的影响

  

# 童年期个性和社会性发展

## 自我意识的发展

### 自我评价能力的特点

+ **自我评价包括:**外表、行为、成绩、运动、社交
+ 社会支持因素对儿童自我评价起着非常重要的作用，其中父母和同学的作用最重要
+ 对自我价值的评价与情感密切联系
+ 小学儿童自我评价与学业经验、同伴交往、自信心等都有密切关系

### 自我控制能力

+ 罗特伯格 通过**延迟满足** 研究儿童自我控制行为
+ 延迟满足 是抑制欲望的及时满足，学会等待
+ 童年期儿童延迟满足能力随年龄增长而显著提高
+ 影响自我控制能力的因素
  + 认知和策略
  + 榜样的作用
  + 家庭教育

## 道德发展

### 道德情感的发展

+ 移情、情感共鸣、内疚、羞愧、良心等
+ 日益丰富，并影响道德行为(保护和培育同理心或爱心，能够比说教更有效地提高人们的道德水平)  

### 道德认知的发展

皮亚杰 的对偶故事

皮亚杰儿童道德认知发展的三个阶段

1. 前道德阶段（0～3岁） 儿童尚不能判断，只能直接接受行为的结果
2. 他律道德阶段（3～7岁）
   + 道德判断的标准受儿童自身以外的价值标准支配
   + 规则规范由权威人士制定，不能改变需严格遵守
   + 行为好坏评定，只看结果、不看动机
3. 自律道德阶段（7岁以后）
   + 道德判断的标准受自己主观的价值标准支配
   + 规则具有相对性，可变
   + 行为好坏评定，看动机或者意图，不是后果

### 道德行为发展

+ 亲社会行为

  定义:对他人有益，对社会有利的积极行为及趋向

  获得需具备的条件

  + 道德动机的发展
  + 逐渐形成能舍身处地为需要帮助者着想的能力
  + 具备亲社会行为的能力

+ 攻击行为

  定义:针对他人的具有敌视性、伤害性或破坏 性的行为。欺负是一种特殊形式的攻击行为

  欺负行为分三种类型：

  + 直接言语欺负
  + 直接身体欺负
  + 间接欺负

​		欺负的特点：

		1. 欺负发生率：小学生欺负发生率20%，随年级升高而下降
		1. 类型发生率：言语欺负出现率最高，其次是直接身体欺负，最低是间接欺负
		1. 性别差异：男生以直接身体欺负为主，女生以言语欺负为主
		1. 对外来的预测：儿童的欺负行为可以预测将来的适应不良



## 童年期同伴交往

同伴交往的重要意义

1. 同伴交往是童年期集体归宿感的心理需求
2. 同伴交往促进儿童的社会认知和社会交往技能的发展
3. 同伴交往有利于儿童自我概念的发展
4. 同伴交往增进良好个性品质和社会责任感

影响儿童在同伴中是否受欢迎的基本因素是儿童本人的**社会交往能力**

儿童的人气特点

+ 不受欢迎的儿童
  + 具有攻击性
  + 对人不友好、不尊重
  + 缺乏合作精神、恶作剧
+ 受忽视的儿童
  + 安静、退缩、顺从

+ 受欢迎的儿童
  + 成绩好，有主见
  + 独立、热情
  + 善于交往、易于合作

友谊对童年期儿童的重要性

+ 朋友为儿童提供学习上的相互帮助
+ 社会交往中的相互支持
+ 情感上的共鸣
+ 提供解决问题和困难的力量
+ 增加快乐和兴趣



儿童对友谊认知

+ 3～5岁【短期游戏伙伴关系】：无友谊概念，一起玩即好朋友
+ 6～9岁【意向帮助关系】：活动行为与自己一致或对自己帮助
+ 9～12岁【双向帮助关系】：相互性，功利性，顺利时合作，不能共患难
+ 12岁以后【对未来预测】：亲密而又相对持久的共享关系，独立性、排他性



选择朋友的因素

+ 相互接近

+ 行为、品质学习成绩和兴趣相近

+ 人格尊重、心理和谐、相互敬穆

  

## 家庭人际关系对童年期儿童心理发展的影响

亲子关系的发展变化

1. 直接交往时间明显减少
2. 父母教养关注重点的转移
3. 父母对儿童控制和儿童自主管理的消长变化



+ 父母控制（6岁以前）：大部分重要决定由父母做出
+ 共同控制（6～12岁）父母在一定距离内监督和引导儿童的行为
+ 儿童控制（12岁以上）儿童自己做出额更多重要决定



儿童期亲子关系的特点

1. 父母与儿童对其行为的共同调节
2. 允许孩子做决定，但是同时监督并指导孩子的决定



- 儿童与父母的交往随年龄的增长而下降

- 与同伴的交往随年龄的增长而快速上升

- 与教师的交往在小学小中年级以前随年

  龄增长而上升，之后一直维持在交往比 率的20%左右



# 错题

童年期儿童的间接推理能力表现在：演绎推理能力、归纳推理能力、类比推理能力

小学阶段儿童概括能力表现的三个阶段为：直观形象水平、形象抽象水平、初步本质抽象水平

在科尔伯特的道德发展理论中，服从和惩罚的道德定向阶段的儿童，特征是 对成人或者规章采取服从的态度，以免受到惩罚

儿童期的思维本质特征是 依赖具体的内容的逻辑思维
