# 幼儿的游戏

游戏是幼儿期的主导活动

+ 幼儿的游戏主导着他们认知和社会性发展
+ 幼儿的各种学习多是通过游戏活动机械能的
+ 游戏是幼儿教育的最佳途径

## 对游戏的理解

+ 古典游戏理论

  着重从游戏与人类关系的角度解释游戏的原因和目的(受进化论影响)

+ 现代游戏理论

  + 精神分析流派

    游戏与儿童**人格和情绪**的发展

    获得现实中不能实现的愿望、处理创伤事件

  + 认知学派

    游戏与儿童**人体认知和发展**的关系

    巩固技能、促进发展

## 游戏的发展

+ 婴儿期：机能游戏
+ 幼儿期：象征性游戏
+ 童年期：规则性游戏

## 象征性游戏的特点

1. 以主题游戏为主
2. 运用以现实物相仿的代替物
3. 通过想象构建虚假的游戏情境
4. 游戏中有创造性
5. 游戏动机在于活动过程

## 游戏的社会性发展

+ 非社会游戏： 独自游戏、旁观游戏
+ 平行游戏：进行相同的游戏活动，但没有相互交流
+ 社会性游戏：协同游戏

## 对儿童心理发展的促进作用

+ 游戏是幼儿活动和情感愉悦的精神寄托
+ 游戏是促进幼儿认知发展和社会性发展的重要渠道
+ 游戏是幼儿之间社会交往的最好园地
+ 游戏是幼儿实现自我价值的最佳载体

# 幼儿的认知发展

## 幼儿记忆的发展

特点：

+ 无意识记忆为主，有意识记忆发展迅速
+ 形象记忆为主，词语记忆逐渐发展
+ 机械记忆和意义记忆同时发展并相互作用

儿童记忆策略的发展：

1. 基本没有记忆策略（5岁前）
2. 经指导能运用记忆策略（5～8、9岁）
3. 主动、自觉地运用记忆策略（10岁以后）

幼儿后期能运用的主要记忆策略

以“不断的口头重复要记住的内容”为主的记忆策略—“复述策略”

1. 视觉“复述”策略
2. 复述策略（说/读）
3. 特征定位策略（特点）

## 幼儿思维的发展

幼儿思维的特点：主要特征是具体形象思维，逻辑思维开始萌芽

1. 思维具体形象的特点（可塑性、动态性）
2. 幼儿认知发展的趋向性
3. 自我中心现象
4. 有一定的计划性和预见性



逻辑思维初步发展

1. 幼儿所提问题类型的变化

2. 幼儿概括能力发展

   + 事物概括的发展：名称概念-》功能-〉内外部特征
   + 类概念的发展-分类能力
     + 一级水平：不能分类
     + 二级水平：能够依据事物的感知特征进行归类
     + 三级水平：依据知识和经验对事物进行归类
     + 四类水平：概念分类

3. 幼儿推理能力的初步发展

   + 幼儿最初的推理是传导推理

     是从表象性象征向逻辑概念过渡的推理形式

   + 对熟悉事物的简单推理

## 幼儿的想象的发展

+ 无意想象经常出现，有意想象日益丰富
+ 再造想象占主要地位，创造想象开始发展
+ 通过良好的教育和训练，幼儿的创造想象得到显著发展
  + 新颖性
  + 神奇性
  + 超越性
  + 未来指向

# 幼儿言语发展

言语发展对儿童发展的重要意义：

1. 语言是儿童人际交流的工具
2. 言语是有助于儿童适应环境的重要工具
3. 帮助儿童超越具体环境，进入新的境界
4. 言语发展是幼儿期心理发展的推动力

## 词汇的发展

+ 词汇数量的增加

一生中词汇量增加最快的年龄段：3～4岁

词汇增长速度：每天2～3个词

3岁左右 1000左右，6岁 3000左右，  7岁4000左右

+ 词类范围的扩大

  实词-》 虚词

  名词-〉动词-》形容词

+ 词义的深化

  + 幼儿期掌握词汇的特点

    1. 词义笼统含糊

    2. 词义所指非常具体
    3. 幼儿末期掌握词汇的概括性逐渐增加

  + 消极词汇（能理解但用不准） 积极词汇（能理解也用的准）的消长

  + 理解和使用词的指标

    理解词的指标：具有间接概括性

    使用词的指标：自发使用，不是自造词、经常使用、有概括意义

## 句子的发展

+ 理解句子的策略
  + 语义策略
  + 词序策略
  + 非言语策略
+ 掌握句法结构的发展
  + 从不完整句发展到完整句
  + 从简单句到复合句
  + 从陈述句到多种形式的句子
  + 从无修饰语发展到有修饰语

## 幼儿口语表达能力的发展

+ 从对话语到独白语
  + 幼儿初期言语表达具有情境性特点
  + 随着年龄增长，对话言语的比重逐渐下降，连续言语的比重逐渐上升，并为独白言语打下基础
+ 情境语句到连贯语句
  + **幼儿期(3-5岁)是口语发展的 关键期**
  + 独白语连贯语的发展是 儿童口语表达能力发展的重要标 志

## 语用技能的发展

语用技能是指个体根据交谈双方的语言意图和所处的语言环境，有效地使用语言工具达到沟通目 的的一系列技能，包括沟通的手势、说的技能和听的技能。

1. 早期沟通的手势
2. 听的语用技能
3. 说的语用技能

+ 对影响有效沟通的情景因素十分敏感(难易环境调整策略)
+ 对同伴的反馈易于做出积极的反应
+ 能够有效地参与谈话(不同对象不同方式、不插话)

# 幼儿个性和社会性的发展

## 个性的初步形成

1. 显示出较明显的气质特点
2. 表现出一定的兴趣爱好差异
3. 表现出一定的能力差异
4. 最初的性格特点表现

## 自我情绪体验的发展

与生理需要相关联的情绪体验 ——》 社会性情感体验发展

1. 幼儿的自尊随年龄的增长而迅速发展

2. 幼儿期自尊水平的高低在一定程度上预测以后的情绪发展和适应性

   



影响儿童自尊的因素

+ 父母的教育方式
+ 同伴关系因素



## 幼儿期儿童认同的发展

认同对儿童发展的影响：

+ 儿童对成人个性品质的效仿称为“认同”;产生认同的基础是儿童知觉到自己与认 同对象之间的相似性或一致性
+ 认同带给儿童归属感和成就感;使儿童获得榜样的力量和发展的动力;对性别意识 和道德意识的发展具有重要影响

幼儿期儿童认同的对象：

通常具有较高地位、具有权威性，有较强的能力，聪明、健壮或漂亮的人

## 儿童发展的第一逆反期

•第一逆反期的表现是幼儿要求行为**活动自主**和**实现自我意志**，反抗父母控制。年龄主要是 3～4岁
