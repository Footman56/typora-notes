# 心理活动的生理学基础

## 神经系统的构造及功能

神经元：组成神经系统的神经细胞，是神经系统的**基本结构**和功能单元，由树突、轴突、细胞体组成

**具有接受刺激，传导刺激的功能**

树突：接受**外界刺激**，并将物理、化学等能量转化成神经冲动；接受上一个神经元的**神经冲动**

轴突：将神经冲动由细胞体提传递给其他神经元

细胞体

髓鞘：绝缘、防止电脉冲在不同轴突之间传递

分成：

+ 感觉神经元（传入）
+ 运动神经元（传出）
+ 中间神经元（联系）

### 外周神经系统（PNS）

把中枢神经与周围器官系统连接起来的神经。主要负责沟通连接

由13对脑神经，31对脊神经

功能：

+ 躯体神经系统（运动）：到达各感觉器官和运动器官的神经
+ 自主神经系统（内在），也叫植物神经系统支配内脏器官
  + 交感神经：负责兴奋
  + 副交感神经：负责平静

### 中枢神经系统（CNS）

包括脑和脊髓。主要控制中心

+ 脑干：最古老的部分
  + 后脑
    + 脑桥
    + 脑延
  + 中脑
  + 脑干网状结构：调节睡眠与觉醒
+ 边缘系统：**负责情绪**
+ 前脑
  + 端脑
  + 间脑
+ 大脑皮层（大脑灰质、大脑皮质）：占40%重量
+ 胼胝体：连接左右脑
+ 丘脑：负责非嗅觉的感觉转换站
+ 下丘脑：调节内脏系统获得的皮层下中枢
+ 小脑：保持身体平衡
  + 随意和不随意的运动
+ 脊髓：传递信息，简单反射

## 高级神经活动的反射学说

巴甫洛夫通过对动物和人的反射 活动的实验研究，发现了许多神经系统高级部位机能活动的规律， 创立了高级神经活动学说。这一 学说也叫条件反射学说，或高级神经活动反射学说

### 兴奋和抑制

巴甫洛夫认为神经活动的基本过程是兴奋和抑制

• 兴奋:神经活动由静息状态或较弱的活动状态，转为活动的状态或较强的活动状态
 • 抑制:神经活动由活动的状态或较强的活动状态，  转为静息状态或较弱的活动状态

### 反射、反射弧和反馈

- 反射是有机体在**神经系统**的参与下，对 **内外环境刺激**做出的**规律性回答**。实现 反射活动的神经通路是反射弧
- 反馈是指反射活动的**结果**又返回传到神经中枢，使神经中枢及时获得效应器活动的信息，从而更有效地调节效应器活动的过程

### 无条件反射和有条件反射

无条件反射-动物和人生而具有的、不 学而会的反射

条件反射-个体通过模仿、学习，在无 条件反射的基础上形成的反射



斯金纳 操作条件反射

小老鼠自己按动按钮后，会获取食物





经典条件反射和操作性条件反射的异同

**相同点:**1都是条件反射;  2实质上都是刺激与刺激的联结;  3规律基本相似;

| **不同点:**|提出者|反应性质|强化时间|学习者角色|学习性质|
| ------------------- | -------- | ----------------- | ---------- | ---------- | --------------------- |
|**经典条件反射**|巴普洛夫|反射性、不随意的|在反应之前|被动学习|与无条件反射 建立联系|
|**操作性条件反射**| 斯金纳   | 自主性、随 意的   | 在反应之后 | 主动学习   | 与特定结果建 立联系   |



**条件反射的抑制**

+ 外抑制：外界中别的刺激更加强烈 
+ 超限抑制：过犹不及
+ 消退抑制：毕业后听到上课铃声时不会再进入教室
+ 泛化现象： 一朝被蛇咬，十年怕井绳
+ 分化抑制：对刺激条件再进行细化。 一声、三声铃声都不会给食物，二声铃声会给食物

**扩散和集中**

**相互诱导**：正诱导（刺激其他行为）和负诱导（抑制其他行为）。同时诱导与相继诱导。

**动力定型**：刺激形成了固定的顺序，反应也跟着形成固定的顺序，只要一个 条件刺激出现，引起一个条件反射，此后的条件反射会一个一个地接连出现， 这就是动力定型。（动力定型是人习惯的生理基础）
