#  感觉 

## 感觉特性

感觉是人脑对直接作用于**感觉器官**的客观事物**个别属性**的反映



感觉可以分为外部感觉、内部感觉。

**感受性**：感觉器官对适宜刺激的感觉能力

**感觉阈限**：是一个范围，能引起 感觉的**最小刺激量**

​	感受性越强，感觉阈限越小，是反比关系

​	感觉阈限 有最小刺激强度～最大刺激强度

​	**绝对感觉阈限** ： 刚刚能引起感觉的最小刺激强度

​	**差别感觉阈限**：刚刚能引起**差别感觉** 刺激的**最小变化量**

## 感觉种类

### 外部感觉

#### 视觉

波长在 **380** 纳米~**780** 纳米之间光波

感觉器官

+ 瞳孔
+ 眼角膜
+ 晶状体
+ 视网膜
  + 锥状体 **明视觉**(中央窝)，分辨细节和颜色
  + 柱状体 **暗视觉**（边缘），分辨轮廓和明暗

##### 颜色特性：

+ 色调：不同的颜色，取决于波长
+ 饱和度：鲜艳程度，取决于灰色占比
+ 明度： 亮暗程度，取决于光强

**灰色只有明度这一特性**

红绿色盲看到的世界没有 红、绿两种颜色，如果是红或者绿的话，眼中是灰色

色觉异常的表现：

• 对明度很敏感，分不清颜色
 • 绝大多数是遗传原因造成的
 • 大约95%的色盲人群是男性色

盲中男女比例为10:1(女<男)

#### 听觉

适宜刺激： 16～20000HZ 

最敏感刺激： 1000～4000HZ

 老年人听觉感受性降低特点:高频→低频→中频

听觉感受器:内耳中的科蒂氏器官

+ 听觉适应：在声音的持续作用下，**听觉感受性**降低的现象。【每个感觉都有感受性】
+ 声音的掩蔽：一个声音由于同时起作用的其他声音的干扰，使听觉阈限升高的现象。（同桌的声音会屏蔽老师的声音）
+ 听觉疲劳：声音强度太大或声音作用时间太长， 引起听觉感受性在**一定时间内降低**的现象。

##### 声音特性

+ 音调： 声波的频率
+ 响度： 声波的振幅
+ 音色： 声波的波形

#### 嗅觉

嗅觉——**最古老的感觉**
适宜刺激:能挥发、有气味的物质 

感受器:鼻腔上部黏膜上的嗅细胞

#### 味觉

适宜刺激:能溶解的、有味道的物质

感受器:味蕾

最基本的味觉:甜、咸、酸、苦【没有辣，辣是知觉，多个感觉的综合作用】

#### 皮肤觉

皮肤感觉:触觉、压觉、振动觉、温觉、冷觉、痛觉

生理零度——皮肤表面的温度

表示触觉灵敏度的指标叫两点阈，手指、面部的 两点阈最小，脊背的两点阈最大。脊背感觉最不明显



### 内部感觉

#### 平衡觉

平衡觉(静觉)，反映了人体的姿势和地心 引力的关系。

感受器:内耳中的前庭器官

前庭觉和视觉相互冲突时，人就会觉得晕

#### 运动觉

运动觉(动觉):反映身体各部分的位置、运动以及肌肉的紧张程度
 感受器:分布在肌肉、筋腱和关节中

#### 内脏觉

内脏感觉(机体觉):包括饥饿、饱胀和渴的感觉，窒息的感觉，疲劳的感觉， 便意、性以及痛的感觉等

**当各种内脏器官处于正常状态时，引不起内脏感觉**

#### 痛觉

痛觉感受器:遍布全身所有组织中

痛觉对机体具有保护性作用

痛觉具有生物学意义

痛觉最难适应



## 感觉现象

### 感觉适应

在外界刺激的持续作用下，感受性**发生变化**的现象

对光适应、对暗适应

### 感觉后像

外界刺激停止作用后，还能暂时保留一段 时间的感觉现象叫感觉后像

- 正后像:感觉后像和刺激物的性质相同 (如:盯着灯泡，看白墙)

- 负后象:感觉后像和刺激物的性质相反， 彩色的负后像是刺激色的补色

  红色的负后像是绿色，黄色的负后像是蓝 色。

### 感觉对比

不同刺激作用于同一感受器官，使感受性发生变化的现象叫感觉对比

同时对比：两种感觉同时发生所形成的对比

相继对比：两种感觉先后发生

### 联觉

一种刺激不仅引起一种感觉，同时还引起另一种感觉的现象，这种现象叫联觉

# 知觉

直接作用于感觉器官的客观物体的**整体**在人脑中的反映

## 知觉与感觉的关系

区别：

+ 感觉只反映事物的**个别属性**，知觉认识了事物的**整体**
+ 感觉是**单一感觉器官**活动的结果，知觉是**各种感觉**协同活动的结果;
+ 感觉不依赖于个人的知识和经验， 知觉受个人的知识和经验的影响。

联系:

知觉来源于感觉，两者都属于对事物的认识

## 知觉特性

### 整体性

知觉具有在**过去经验**的基础上， 把物体的各个部分、各种属性**结合**起来，成为一个**整体**的特性

### 选择性

人们根据**感觉通道的容量和自己的需要**，把一部分物体当做知觉的对 象，而把其他对象当做**背景**，也就 是**有选择**地知觉外界物体

### 恒常性

在一定范围内，知觉的条件发生了 变化，而知觉的**映像却保持相对稳定不变**的知觉特性

### 理解性

在知觉外界物体时，人们总要用 **过去的经验**对其加以解释，并用 词语揭示其特性

## 知觉种类

### 空间知觉

对物体的大小、形状、距离、方位等空间特性的知觉

+ 大小知觉

  由物体在**视网膜**上形成的视像的大小、 物体与观察者之间的**距离**以及周围**参照物**等因素决定

+ 形状知觉

  **视网膜像**提供了视觉信息，视线沿**物体边界**的扫描运动提供了动觉信息，手的 触摸提供了**触觉信息**，这些信息的结合 形成了形状知觉 。形状知觉 = 视觉 + 触觉

+ 方位知觉：自身、双耳听觉提供的信息

+ 距离知觉：判断距离远近的知觉，又叫**深度知觉、立体知觉**

  判断距离的线索

  + 肌肉运动线索（眼睛的调节作用、双眼视轴辐合）
  + 单眼线索：对象的重叠、透视、明暗、阴影、运动视差
  + 双眼线索：**双眼视差是形成 深度知觉的最主 要线索**

### 时间知觉

对物质现象的**延续性**和**顺序性**的知觉

线索：计时器提供的信息、自然界昼夜 交替、四季周期性变化、人体生理活动、心理活动周期性变化

机体生理节律→行为节律→**生物钟**

估计时间的准确性: 听觉  > 触觉 > 视觉

### 运动知觉

对物体在**空间位置中的位移**产生的知觉

+ 似动现象（静止的物体看似运动）
+ 诱导运动 （一动一静）
+ 自主运动 （本来没有动，自己感觉动了）

### 错觉

在**特定条件**下产生的，对客观事物的歪曲知觉，这种歪曲往往带有**固定的倾向** 

**只要具备错觉产生的条件，错觉必然会产生，通过主观努力无法克服**

不同感受器间相互作用:形重、视听





# 错题

人类听觉的适宜频率是 16～20000HZ

色觉有缺陷的人 无法通过 饱和度的差别 辨认颜色

内行看门道，外行看热闹 体现了知觉的理解性

感觉器官对 适宜刺激的感觉能力叫做 感受性

色盲患者 一般也能”辨认“眼色

绿叶陪衬下红花看起来表现更红 是 同时比对现象，颜色对比现象

痛觉反应一个人的意志特点、对机体有保护作用、不能反应一个人的灵敏程度、痛觉最难适应

不同感觉器官之间相互形成的错觉是 视听错觉、形重错觉

人估计时间的方式：心理活动的周期性变化、各种计时工具、生理活动的周期性变化

声音的特性包括音调、音色。不包括频率。响度-振幅、音调-频率、音色-波形
