# 记忆

记忆是 **过去的经验**在头脑中的反映

过去的经验是指，过去对事物的感知、对问题的思考、对某个事件引起的情绪体验，以及进行过的动作操作

## 记忆的种类

### 按内容

+ 形象记忆
+ 情景记忆
+ 情绪记忆
+ 语义记忆：语词-逻辑记忆 用**语词**概括 有组织的知识
+ 动作记忆

### 按是否意识到

+ 外显记忆：在意识的控制下，过去的经验对当前作业产生的有意识的影响，又称**受意识控制**的记忆
+ 内隐记忆：个体并没有意识到，过去的经验却对当前的活动产生了影响，又称自动的、无意识的记忆

### 按认知

按信息的编码、储存和提取的方式的不同，以及信息储存时间长短不同

+ 瞬时记忆（感觉记忆） 保持时间 < 1秒  【编码方式- 事物的形象】  容量大
+ 短时记忆 （工作记忆） 保持时间 < 1 分  【编码方式- 语言文字听觉、非语言文字-形象】  容量 5～9 单位
+ 长时记忆 保持时间 > 1 分 【编码方式：语义编码、形象编码 】 容量无限

## 记忆过程

1. 识记：学习与取得知识和经验的过程

2. 保持：知识和经验在大脑中储存和巩固的过程

3. 再现、再认

   再现--从大脑中提取知识和经验的过程（简答题）

   再认--识记过的材料不能再现。但在它重现时却能有一种熟悉感并能确认是自己接般过的材料（选择题）

**再现**记住的内容更加牢固

## 遗忘及遗忘的规律

遗忘：对**识记过材料**的**不能回忆**，**不能再认**，或者发生错误的回忆和再认

遗忘的规律：**先快后慢**

## 遗忘的原因

1. 自然衰退

   时间推移，随着时间推移，记忆的痕迹越来越淡薄，最终导致遗忘

2. 干扰

   + 前摄抑制 ： 最开始学的知识会影响后面的内容（首因效应）
   
   + 后摄抑制：后慢学的知识影响之前学的内容（近因效应）
   
     学的知识在 开头 、结尾记住的最牢固。

# 思维

思维是人脑**对客观事物的本质**和**事物之间的内在联系**的认识，思维作为一种反应形式

## 思维特征

+ 间接性：从已知推出未知
+ 概括性：对**一类事物共同属性**抽取总结

## 智力操作过程

1. 分析
2. 综合
3. 抽象
4. 概括

## 思维种类

### 按形态划分

+ 动作思维：以实际动作为支柱
+ 形象思维：是以直观形象和表象为支柱
+ 抽象思维：用词进行**判断**、**推理** 并得出结论，又叫词的思维或逻辑思维

### 按搜索答案

+ 辐合思维：按照已知信息和熟悉的 规则进行的思维 得出正确答案，比如说明书
+ 发散思维：沿着不同的方向探索 问题的答案的思维 头脑风暴的过程

### 按是否有创造性

+ 创造性思维：用**独创的方法** 去解决问题的思维
+ 再造性思维：用**已知的方法**去 解决问题的思维

## 概念

**概念是思维活动的结果和产 物**，是思维活动借以进行的单元。个体掌握概念 是由浅到深的，个体掌握概念的层次反映了他的 思维发展水平。

+ 内涵概念：概念所包含的事物的**本质属性**:质
+ 外延概念：概念所包含的**范围**:量

**内涵与外延成反比**

## 问题解决因素

在问题空间中进行搜索，以便从问题的初始状态达到目标状态的过程

+ 迁移： 将一个问题转换成熟悉的问题
+ 原型启发： 根据一个事物参考
+ 定势：从事某种活动前的**心理准备**

言语：人们运用语言进行交际的过程，**心理现象**

## 言语形式

# 表象与想象

表象：头脑中所出现的事物的形象

想象：运用已有的表象对其进行加工和改造，创造出**新形象**

表象的**可操作性**，使表象可以 成为想象的素材

## 想象分类：

+ 无意想象：没有预定目的，在某种刺激作用下不由自主产生的想象

+ 有意想象：在一定**目的、意图和任务**影响下 有意识进行的想象

  + 创造想象：不依据现成描述和图示独立创造出**新形象**
  + 再造想象：根据**语言描述或图表模型**示意，在头脑中形成相应形象的过程 如读者
  + 幻想：和一个人的愿望相联系并指向未来的想象
