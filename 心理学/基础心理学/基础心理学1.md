# 基础心理学

##  研究对象

基础心理学是以**正常成人**的**心理现象**为研究对象，总结心理活动**最普遍、最一般**规律的心理学基础科学

## 研究内容

分为心理过程、心理特性

心理过程：

+ 认知过程： 人认识外界事物的过程,对作用于人的感觉器官的外界事物进行**信息加工**的过程。包括：感觉、记忆、思维
+ 情绪和情感过程：情绪和情感是伴随认知和意志过程产生对外界事物的**态度**的体验。是人脑观察外界事物于**主体需要**之间的关系反映
+ 意志过程：意志是有意识的确定目的、调节和支配行为、克服困难、达到预期目标的心理过程。受意志支配的行动叫意志行动

心理特性：

+ 心理动机： 人对客观事物的态度以及对活动事物的**选择和趋势**，是人从事心理活动的**指向性**和**内部动力**，包括需求、价值观、世界观
+ 人格特征：人在心理过程中形成的**稳定而经常出现**的心理特性，集中反映一个人的心理面貌。包括能力、气质、性格



**心理是脑的机能**，灵长目处于思维的萌芽阶段，**条件反射是心理现象产生的标志。**

# 心理学发展史

盖伦提出了“气质“

希罗克拉底提出四种气质：胆汁质、粘液质、抑郁质、多血质

科学心理学诞生标志：1879年 冯特在莱比锡大学建立世界上第一个心理学实验室

## 古典主义

### 构造主义

创始人：冯特、铁钦纳

主张：采用**内省实验**的方法，分析**意识**的内容，并找出**意识的组成部分**。只问意识经验有什么**元素**构成，不问意识内容的来源、意义和作用

贡献：**开创现代科学心理学**并为其发展奠定了基础，

但是脱离现实、内省法过于主观、可重复性差

### 机能主义

创建人：杜威、詹姆斯、安吉尔

主张：反对把意思分解，主张意识是 **连续的整体**

反对把心理看做一种不起作用的副现象，强调心理的**适应功能** ； 反对把心理学只看作一门纯科学，**重视心理学的实际应用**； 认为心理学应该研究心理在适应环境中的**机能作用**

贡献：大大推动了心理学在实际领域中的发展，促进了 教育心理学的发展，还推动了工业心理学的发展，特别 是对工作环境中人的因素研究。并为**行为主义**心理学开 辟了道路。



### 行为主义

创建人：华生、斯金纳、班杜拉

主张：心理学要抛开意识，径直**研究行为**

心理学研究行为的任务，就是查明 **刺激与反应**之间的规律性关系。只要确定**刺激和反应**(S-R)之间的关系，就可以预测行为，并通过控制环境去塑造人的心理和行为。

贡献:以客观的、可以验证的实验方法，对科学心理学 的发展作出了很大贡献，尤其在学习的规律，条件反射、 奖励和惩罚的应用、行为矫正等方面极具价值。

局限:由于排斥内省法及对人的意识经验的研究，使心 理学的研究内涵窄化，从而阻碍了心理学的健康发展。



### 格式塔

创建人：魏特海默、克勒、科夫卡

主张从**整体上** 来研究心理现象

贡献:在知觉、问题解决、思维、学习顿悟、 艺术、逻辑、政治等研究方面具有很大贡献， 促进了认知心理学的兴起，也成为心理治疗重 要理论基础。



### 精神分析

创建人：弗洛伊德、埃里克森

用潜意识、生本能、死本能、力比多等观念来解释人的行为动力

心理结构:意识;前意识;潜意识。

人格结构三个层次——本我、自我、超我并以焦虑和心理防御机制解释三个“我”之间的矛
 盾冲突。

贡献：弗洛伊德对**精神病学**和**临床心理学**的影响是深远的， 对科学心理学也有相当重要的意义。

局限：1.六岁定一生太强调早期经验对人发展的影响，没有注意到环境的作用也是很强大的。2.研究方法缺乏科学 的严谨性;3.过分强调无意识，并将其与意识对立;4.早期 观点有泛性论的特点，把性欲夸大为支配人类一切行为的 动机。



## 当代心理学

### 人本主义

创始人：罗杰斯、马斯洛

人有自我的纯主观意识,有**自我实现**的需要。只要有适当的环境, 人就会努力去实现自 我、完善自我,最终达到自我实现。所以,人本 主义重视人自身的价值,提倡充分**发挥人的潜能**

求助者中心疗法、需求层次理论

贡献:以人的社会性特点对人的心理本质进行的新 诠释，对教育心理学、发展心理学、心理咨询与辅 导、心理治疗等产生了很大的影响，第三势力。

局限:人本主义心理学家的大多数用语具有模糊性 和主观性，很难对其下操作性定义。

### 认知心理学

创始人：奈瑟尔、皮亚杰、贝克

它把人看做是一个类似于计算机的信息加工系统,并以信息加工的观点, 即从信息的输入、编 码、转换、储存和提取等的加工过程来研究人 的**高级心理过程**, 主要是注意、感知觉、 表象、 记忆、思维和语言等**认知活动**

### 生理心理学

生理心理学是研究心理现象产生的**生理过程**的心理学分支。 生理心理学是心理活动的生理基础和脑的机制



# 研究心理学原则

+ 客观性
+ 辩证发展
+ 理论联系实际

# 研究心理学方法

+ 观察法：在自然条件下,有目的、有计划地系统观察人的 观察法 行为和活动,从中发现心理现象产生和发展的规

  律的方法叫观察法,或叫自然观察法

+ 调查法：就某一问题，用口头或书面的形式向被调查的对象提问， 调查法 让他回答，通过对他的回答的分析来了解他的心理活动

  的方法叫调查法(问卷/访谈)

+ 个案法：对某一被试者所做的多方面的深入详细研究，包括他的 个案法 历史资料、作业成绩、测验结果，以及别人对他的评价等，目的在于发现影响某种心理和行为的原因

+ 实验法：主试者在严格控制的条件下,观察被试者的行为或活动，探索客观条件和人的心理活动之间因果联系的研究方法





# 错题

1. 持环境决定论的是 机能主义，不是行为主义
2. 心理活动的器官是神经系统
3. 说法正确的是：
   + 认知是人认识外界事物的过程
   + 人格可以通过心理过程得到表现
   + 情感属于心理过程

**意志不是需要克服困难和挫折的行为**。意志包括多个过程，（确定目标、行动、克服困难、达成目的）
