

# 意识

意识是人脑特有的反映功能，是人心理和动物心理的**根本区别**，是自然进化的最高产物；在**觉醒状态**下的擦觉，或者叫觉知

两种特殊的意识状态：梦、睡眠

根据脑电波的变化，睡眠分成四个阶段

+ 睡眠
  1. 入睡期
  2. 浅眠期：会产生睡眠锭：短暂爆发、频率高、波幅大
  3. 熟睡期
  4. 深睡期

```
β波-清醒时出现
α波-安静和休息时
θ波-浅睡时出现
δ波-深睡时出现
```

+ 梦

**发生在快速动眼期**

梦的特点：不连续、不协调、认知的不确定

梦是一种正常的心理现象和生理现象

睡眠时人人都会做梦

梦剥夺会影响人体健康

弗洛伊德：梦是压抑到潜意识中的冲动和愿意反映

# 注意

注意是心理活动或者意识活动，对**一个对象的指向和集中**

**注意是一个伴随状态**

## 注意的种类

+ 无意注意：没有意识的
  + 强度大、对比鲜明、突然出现、变化运动、新颖、自己感应趣的会更容易吸引
+ 有意注意：
+ 随意后注意：自动化行为-吃饭

## 注意广度

同一时间内，意识所能清楚把握对象的数量叫做 注意广度， 即 5～9项目

## 注意稳定性

对选择对象注意能**稳定保持** 多长时间的特性

注意的起伏：在稳定注意的条件下，感受性也会发生 **周期性地增强和减弱**的变化现象。

注意的分散：注意离开 了心理活动所要指向的对象，而 被**无关的对象**吸引去的现象。 

## 注意转移

由于**任务的变**化，注意由一种对象转移到另一种对象 。

注意转移和注意分散的区别是 **是否有意识**

## 注意分配

**同一时间**，把注意指向不同的对象，**同时从事**几种不同的活动

注意分配的条件：

+ 所从事的活动中有一些活动是非常熟练的
+ 所从事的几种活动之间有内在的联系
+ 两种活动不在**同一感觉道**，不用同一种心理操作完成。（边上课边吃东西）

# 需要

有机体内部**不平衡**状态，表现为有机体对内外条件的需求 

+ 自然需要：饮食【人和动物有本质区别：对象与方式】
+ 社会需要：社交【人特有的】
+ 物质需要：生活条件
+ 精神需要：文化

## 需求层次理论

马洛斯需求层次图

+ 缺失性需要：
  1. 生理需要： 水、空气的需要
  2. 安全需要： 对安全、秩序、稳定、免除恐惧和焦虑的需要
  3. 爱和归属需要： 要求与他人建立情感联系、隶属于某个组织、并在全体中享受地位
  4. 尊重需要：希望有稳定的地位、受他人尊重、并尊重他人
+ 生长需要
  + 自我实现需要：最大限度发挥自己的潜能

只有较低层次的需要得到基本的满足， 较高层次的需要才会出现。

层次越低的需要出现得越**早**，层次越高 的需要出现得越**晚**。

层次越低的需要力量越强。

会有一些特殊情况

# 动机

激发个体朝一定的**目标**活动，并维持这种活动的一种内在心理动力

**需要是行为的原动力、动机是行为的直接动力**

## 动机的产生

动机是在需要的基础上产生

+ 内驱力：由**生理需要**引起的，推动个体为恢复机体内部平衡的唤醒 状态。——生理性动机
+ 诱因：能引起有机体的定向活动，并能满足某种需要的**外部条件**。
+ 情绪也具有动机的作用

## 动机与行为之间的关系

同一行为可以由不同动机引起，不同活动也可以由相同的或相似的动机引起。

一个人的活动动机是多种多样的，有些动机起着主导的作用， 有些动机则处于从属的地位

动机和效果之间一般是**一致**的。

动机与解决问题的效率是 **倒U型曲线**

# 情绪

情绪和情感是人对客观外界事物的 **态度的体验**，是人脑对客观外界事物与主体需要之间关系的反映

功能：

+ 适应：点头微笑，眼神和善
+ 动机:害怕了就要跑
+ 组织:积极情绪对学习有帮助
+ 信号:传递信息，沟通思想，如特工

情绪和情感的外部表现形式——**表情**

表情分为：

+ 面部表情
+ 身体表情
+ 言语表情：说话的音调、速度



## 两极性

情绪和情感的变化有**不同的维度**，情绪和情感在每一维度上的变化都具有两极对立的特性

+ 动力性：
  + 增力
  + 减力
+ 强度：
  + 强
  + 弱
+ 激动度
  + 激动
  + 平静
+ 紧张度
  + 紧张
  + 放松

##  情感种类

+ 基本情绪
  + 喜
  + 怒
  + 哀
  + 惧
+ 复合情绪：由基础情绪组合而成



按情绪的状态(情绪发生的速度、强度和持续时间的长短划分

+ 心境：微弱、持久具弥散性
+ 激情：强烈、爆发式的  持续时间较短 由重大的突如其来的事件或激烈的意向冲突引起
+ 应激：出现意外事件或遇到危险情景时出现的高度紧张的情绪状态



# 意志

有意识地确立目的，调节和支配行 动，并通过克服困难和挫折， 实 现预定目的的**心理过程**

【意志行动的基本阶段】

1. 准备阶段:好几个目的怎么办
2.  执行决定阶段

## 意志冲突形式

+  双趋式冲突：【鱼和熊掌不可兼得】两个具有同样吸引力的目标，两个动机同样强烈，但条件限制，只能达到一个目标，而不能同时达到所有的目标。
+ 双避式冲突：【两其相害取其轻】两个目标都想避开，但只能避开一个目标的时候，人们只好选择对自己损失小的目标，避开损失大的目标
+ 趋避式冲突：【有利有弊】想获得一个目标，它对自己既有利，又有弊时，所遇到的矛盾心情。
+ 双重趋避式冲突：【多个都是有利有弊】如果有多个目标，每个目标对自己都有利，也都有弊，反复权衡拿不定主意时的矛盾心情。



## 意志的品质

+ 自觉性：【主动】与之相反的是容易受暗示型武断从事
+ 果断性：【迅速】与之相反的是优柔寡断、鲁莽草率
+ 坚韧性：【毅力】与之相反的是虎头蛇尾、执拗
+ 自执性：【自律】与之相反的是怯懦、任性
