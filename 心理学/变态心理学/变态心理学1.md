# 精神分析

精神分析解析异常心理：

+ 心理过程主要是潜意识，意识的心理过程则是整个心灵的分离部分和动作
+ 性的冲动，是神经症和精神疾病的重要起因

有如下推论：

1. 人类的生物本能 是心理活动的动力，动力叫 “力比多”

2.  “力比多” 在幼年期 驱动人的性心理发展，分成三个阶段

   + 口欲期（0-1岁）：喂奶过多产生依赖，喂奶过少产生孤立
   + 肛欲期（1-3岁）：对规则的遵守
   + 生殖器器（3-5岁）： 性别的认识

   **合理的度过“性心理”发展的每个阶段，是未来心理健康的充分和必要条件**

3. 人的心理结构由 潜意识（本我）、前意识（自我）、意识（超我）构成
4. 本我 按照 “快乐原则” ；超我 是按“道德原则”； 自我按照 “现实原则”
5. 在自我受到超我、本我压抑时发展出的一种机能，叫做防御机制
   + 合理化：啊Q精神
   + 拒绝： 不接受
   + 投射： 看谁都这样
   + 升华：合理的宣泄

认为人的异常心理是一下原因造成的

+ 性心理发展中 接受的刺激过多或者多少，造成性心理退化，性心理 “**固着**”

+ 自我不能妥善处理  本我 与超我之间的冲突，感到**焦虑**

+ 为防止、抵御和消除焦虑，需克制、**压抑**非理性的力量。如果压抑力量不足，使内部冲突 /非理性冲动冲破防线，就会产生异常行为

  如果冲不破防线，会形成潜意识的痛苦，以梦、口误或失误行为表达

# 行为主义

巴普洛夫通过**实验结果的分析**说明异常心理

通过**动物实验**，并由此判断出**高级神经系统功能的病理生理**机制后，对临床病人进行观察，最后用**类比**的方法解释人的异常心理

神经症和精神病的区别是 神经活动障碍的复杂性和精细特征上的区别

**兴奋和抑制** 这两个基本神经过程冲突造成的疾病

引发机能性神经障碍的两个条件

+ 兴奋过程和抑制过程的冲突
+ 强有力的、异乎寻常的刺激


神经衰弱-兴奋过程的优势和抑制过程的薄弱

癔症- 兴奋过程的薄弱和抑制过程的优势

# 人本主义

人本主义提出 “潜能” 潜能具有 趋向完美的性质和特点

心理异常就是“潜能”趋于完善的特征受到阻碍、是自我无法实现的结果

马洛斯 认为心理异常 最基本的 表现就是 “存在焦虑”，是存在和责任的冲突



# 正常的心理

正常的心理 具有如下功能：

1. 保障人顺利的适应环境，健康生存发展
2. 保障人正常进行人际交往，肩负责任，使社会组织**正常**运行
3. 保障人**正常**地反映、认识客观世界的本质及其规律



# 正常、异常心理标准

##  标准的区分

李心天 1991 提出标准

+ 医学标准：重视物理、化学检测
+ 统计学标准：显示正态分布，远离两端的视为异常
+ 内省经验标准：病人的主观体验、观察者的经验
+ 社会适应标准：是否按照社会生活的需要适应环境。正常异常主要与行为 的社会常模比较

## 心理学区分原则

郭念峰

根据 心理是客观现实的反映，是脑的机能

+ 人格的相对稳定性原则
