# 注意障碍

**注意不是 独立的心理过程，是一个状态，**注意对判断是否有意识障碍 有重要意义，**意识障碍时伴随注意障碍**

## 注意减弱

主动和被动注意的兴奋性减弱

## 注意狭隘

患者的注意范围显著缩小，主动注意减弱，见于意识障碍、激情状态、专注状态、智能障碍患者

# 记忆障碍

## 记忆增强

病理性的，表现为不重要的事情回忆，重要的记不住

## 记忆减退

远记忆力（很久的事情记不起来） 和 近记忆力 （刚发生的事情记不起来） 减退

## 遗忘

局限于**某一事情**或者**某一时期**的内容经历不能回忆

#### 顺行性遗忘

忘记疾病发生之后一段时间内经历的事情

#### 逆行性遗忘

忘记疾病发生之前一段时间内经历的事情



遗忘对脑外伤性精神障碍的诊断有参考价值

#### 心因性遗忘症

心理因素引起的遗忘

#### 错构

记忆的错误，对过去的经历，在时间、地点情节上出现错误，并坚信不移

#### 虚构

虚构的事实来弥补他所遗忘的经历

# 智能障碍

## 精神发育迟滞

先天或者围产期或者生长发育成熟以前 ，由于多种因素，导致大脑发育不全，智能停留在一阶段，

## 痴呆

综合症，意识清楚下后天获得记忆、智能受损。

**临床上大多数痴呆是脑器质性的**

# 自知力

患者对自身精神病态的认识和批判能力

自知力是精神科 用来判断患者是否有精神障碍，精神障碍严重程度，以及疗效的重要指针之一



# 情绪障碍

## 以程度变化为主的情绪障碍

### 情绪高涨

精力充沛、内心充满幸福感、睡眠减少、自我评价过高、易激怒、有感染力

### 情绪低落

自我评价过低、有自杀倾向、面带愁容

### 焦虑

在缺乏充分的事实根据和客观因素下，感到不安

### 恐怖

心悸、气短、尿频、出汗、四肢震颤 等自主神经功能紊乱 

遇到特定境遇会产生恐惧心理

|      | 原因                 | 对象 | 特点                           |
| ---- | -------------------- | ---- | ------------------------------ |
| 焦虑 | 预感的、不明确的危险 | 模糊 | 与未来相联系、是心理矛盾的结果 |
| 恐怖 | 已知的、明确的危险   | 具体 | 针对当前，不是心理矛盾的结果   |



## 以性质变化为主的情绪障碍

#### 情绪迟钝

对一般能引起鲜明情绪反应事情反应平淡，缺乏相应的情绪反应

#### 情绪淡漠

对能引起正常人情绪波动的事情与自身切身利益的事情缺乏相应的反应，更加严重

#### 情绪倒错

反应与现实刺激的性质不相称、情绪反应与思维内容不协调

## 脑器质性损伤的情绪障碍

### 情绪脆弱

因细小的无关重要的事情伤心或者兴奋

#### 易激怒

因细小的事情引起强烈的反应

#### 强制性哭笑

突然出现不能控制的面部表情

#### 欣快

在痴呆基础上 一种“情绪高涨”

# 意志行为障碍

## 意志增强

意志活动的增多

## 意志缺乏

缺乏应有的主动性和积极性

## 意志减退

意志活动减少

+ 抑郁状态
+ 程度较轻的意志缺乏

## 精神运动性兴奋

分为协调性和不协调性

协调性：行动与思维、情绪协调一致、有目的的

不协调性：行动与思维、情绪不协调一致、目的不明确

## 精神运动性抑制



### 木僵

不言不语、不吃不喝、不动

### 违拗

### 蜡样屈曲

肢体任人摆布

### 缄默

不能说话、缄默不语、不能回答问题

### 被动性服从

被动地服从医生或者其他人的命令

### 刻板动作

重复单一的动作

### 模仿动作

无目的的模仿别人的动作

### 紧张症性综合症

紧张性木僵和紧张性兴奋 单独或者交替出现

### 意向倒错

患者的意向活动与一般常青相违背

### 作态

做出奇怪的动作

### 强迫动作

违反本人意愿、反复出现的动作





# 精神分裂症

是一组 病因未明 的常见精神障碍，具有感知、思维、情绪、意志和行为多方面的障碍、以精神活动的不协调和脱离现实为特征

## 特点

+ 直接病因未明、诱发因素诸多
+ 无意识障碍、无智能障碍，部分病人出现认知障碍
+ 青壮年起病
+ 缓慢起病、病程迁延
+ 部分患者慢性衰退、病情反复
+ 发作期自知力基本丧失

## 偏执型

又称妄想型 ， 发展成**妄想**

## 单纯型

幻想妄想不明显，孤僻、懒散

## 青春型

较常见，发病于青春期，思维内容离奇，常有兴奋冲动行为

## 紧张型

出现木僵状态，



# 妄想性障碍

又叫偏执性精神障碍

出现单一 或者一整套相关的妄想，这种妄想是持久的，不荒谬

特点

+ 病因不明
+ 病前性格：固执、主观、敏感、好强
+ 个性缺陷 + 心理社会应激因素 



# 急性短暂性精神障碍

+ 在两周内 急性起病
+ 以精神病性症状为主
+ 病前有相应的心因
+ 在2～3月可完全康复





# 错题

精神迟钝 缺乏反应，但没有到丧失的程度，情绪淡漠 达到丧失程度

**偏执性精神障碍 又叫 妄想性 障碍**

弗洛伊德认为 道德性焦虑中危险源于 超我

神经症者多主动求医，有认知障碍的不会主动求医

破裂性思维 仅保留词语之间的关系，句与句之间的联系没有

患者在 意识障碍的情况下出现语词杂拌  这种症状是 思维不连贯

弗洛伊德认为 神经性焦虑的特点包括：意识不到、威胁来自本能

刻板动作：重复某一动作、伴随刻板言语，感受不到内心痛苦

科萨科夫综合症包括：记忆减退、虚构、定向力障碍、错构

思维形式障碍：思维奔逸、思维中断；**思维形式障碍不包括思维鸣响**

弗洛伊德将焦虑分为： 客体性焦虑、道德性焦虑、神经性焦虑
