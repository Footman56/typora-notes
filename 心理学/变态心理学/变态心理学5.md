## 进食障碍

一组以进食行为异常为主的精神障碍

多见于青少年女性的进食异常，特征为故意限食，

### 神经性贪食

### 神经性呕吐

自发或者故意诱发反复呕吐特征

## 睡眠障碍

各种心理社会因素引起的睡眠和觉醒障碍，包括失眠症、嗜睡症、某些发作性睡眠异常情况

失眠症是一种失眠为主的睡眠质量不满意状态，嗜睡症是白天睡眠过多

# 癔症

又称**歇斯底里**，没有器质性质的病变，以人格倾向为基础，在心理社会环境因素影响下产生的精神障碍

临床表现为：感觉障碍、运动障碍、意识改变状态，具有做作、夸大富有情感色彩等特点，由暗示诱发，

## 转换性障碍（癔症性躯体障碍）

表现为**运动障碍**和**感觉障碍**，其特点多种检查不能发现神经系统和内脏器官相迎的病变

## 一般心理问题

+ 由于现实生活、工作压力 处事失误产生内心冲突，体验到不良情绪

+ 持续满一个月，两个月无法自行化解

+ 不良情绪反应仍在，始终能保持行为不失常态，

+ 不良情绪的激发因素仅局限最初事件

  

## 严重心理问题

+ 较为强烈、对个体威胁较大的现实刺激
+ 痛苦情绪间断或者不间断持续两个月以上，半年一下
+ 遭受刺激强度越大，反应越强烈，影响正常生活
+ 相关联的刺激也能引起痛苦

# 神经症

神经症是一种精神 障碍，表现为持久的心理冲突

特点：

1. 意识的心理冲突
2. 神经痛苦
3. 持久性
4. 神经症妨碍病人的心理功能和社会功能
5. 没有任何器质性病变为基础

心理冲突分为常形与变形

常形：

1. 他与现实处境直接相联系
2. 带有明显的道德性质

变形：

1. 与现实处境没有什么关系，无法平静面对鸡毛蒜皮的小事
2. 不带明显的道德色彩

# 健康心理学

健康心理学是 心理学借助“现代医疗模式”，主动介入医学领域的结果

保健、诊病、防病、治病

常见躯体疾病患者的心理问题

+ 对客观世界和自身价值观念的态度发生改变
+ 把注意力从外界转移到自身的体验和感觉上
+ 情绪低落
+ 时间感觉发生变化
+ 精神偏离日常状态

# 压力与健康

压力应当是一种经验到的东西。压力是压力源和压力反应共同构成的一种认知和行为体验过程

压力源：现实生活要求人们去适应的事件

压力反应：主体察觉到压力源后，出现心理生理行为反应

## 压力源种类

+ 生物性压力源
+ 精神性压力源
+ 社会环境压力源
  + 纯社会性的
  + 由自身状态造成的人际适应问题

压力源的测评

+ 社会在适应量表
+ 日常生活中小困扰的测量
+ 知觉压力的测评



## 压力的适应

按强度分成三类

+ 一般单一性生活压力
+ 叠加性压力
  + 同时性叠加压力（四面楚歌）
  + 继时性的叠加压力（祸不单行）
+ 破环性压力（极端压力）



## 灾难症侯群

强大的自然灾害后的心理反应

1. 惊吓期 （失魂落魄）
2. 恢复期（后怕）
3. 康复期



## 一般适应症侯群

1. 警觉阶段
2. 搏斗阶段
3. 衰竭阶段



## 压力的临床结果和中介系统



从压力源到临床相的逻辑过程

1. 压力的响应阶段
2. 中介系统的增益或消解过程
   + 认知系统
     + 认知、评估作用
     + 调节控制作用
       + 行为的自我控制
       + 认知的控制
       + 环境的控制
   + 社会支持系统
     + 具体地支持当事人
     + 给当事人精神支持
   + 生物调节系统（最主要的免疫系统）
     + 神经内分泌系统
     + 免疫系统
3. 临床相阶段
   + 及时型的症状
   + 滞后型的临床相



# 错题

人对突然的强大精神刺激的抵抗能力叫心理活动强度

健康心理学侧重研究 伤害身体健康的生活的方式和心理活动

个体对事件的实际反应取决于经过中介系统的增益或者消解后，压力的相对强度

一般适应症侯群 搏斗阶段：一点小事就能引起强烈情绪

日常生活小困扰量表提出：日常小压力比主要的社会改变更能预测健康

赛利提出的压力适应的三个阶段统称 一般适应症侯群 

良好的生物调节系统功能可以防止或者降低应激引起的 躯体化症状

破环性压力可以造成：急性应激障碍、创伤后应激障碍

健康心理学的工作领域包括：促进和维护健康的心理学问题、疾病患者的心理学问题、促进健康服务和健康服务政策的制定
