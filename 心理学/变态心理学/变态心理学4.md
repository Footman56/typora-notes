# 心境障碍

旧称 情绪性精神障碍，是以明显而持久的情绪高涨或者情绪低落为主的一组精神障碍

特点

+ 心境变化通常伴随整体活动水平的改变
+ 其他症状多继发于上述改变
+ 有精神病性症状 即 幻觉、继发妄想
+ 大多有反复发作倾向
+ 每次发病常常与应激性事件或者处境有关

## 躁狂发作

特点： 情绪高涨、思维奔逸、精神运动性兴奋

发作形式：轻型躁狂、无精神症状躁狂、有精神症状躁狂、复发性状躁狂

以情绪高涨或者易激怒为主要特征。且症状持续一周，有以下3个症状

+ 言语比平时显著增多
+ 联想加快
+ 注意力不集中
+ 自我评价过高
+ 自我感觉良好
+ 睡眠需要量减少且不感疲乏
+ 活动增多，或精神性运动兴奋
+ 行为轻率
+ 性欲明显亢进
+ 社会功能受损

## 抑郁发作

特点：情绪低落、思维缓慢、语言动作减少和迟缓

发作形式：轻型抑郁症、无精神症状抑郁症、有精神症状抑郁症、复发性状抑郁症

诊断标准：以心境低落为主要特征，持续两周，至少有4项

+ 对日常生活丧失兴趣、无愉快感
+ 精力明显减退，无原因持续疲乏感
+ 精神运动性迟缓或者激越
+ 自我评价过低
+ 联想困难
+ 反复出现想死的念头
+ 失眠、早醒
+ 食欲不振，体重明显减轻
+ 性欲明显减退

## 双相障碍

表现情绪高涨与情绪低落交错发作

# 持续性心境障碍

持续性并伴有起伏的心境障碍，每次发作极少严重到轻度躁狂或者轻度抑郁

发作形式：环性心境障碍、恶劣心境

# 应激相关障碍

又名反应性精神障碍或则心因性精神障碍：值一组主要由心理、社会因素引起的异常心理反应而导致的精神障碍

## 急性应激障碍

急剧严重的精神打击，主要表现 意识障碍、意识范围狭隘、定向障碍、对周围感知迟钝，有强烈恐惧、精神性运动性兴奋或则抑制

诊断要点：

+ 有应激事件
+ 主要表现为意识障碍，
+ 持续时间不长

## 创伤后应激障碍

又称延迟性心因性反应，遭受强烈的或者灾难性精神创伤事件之后，**延迟出现**，长期持续的精神障碍

+ 反复发生的创伤性体验重现，梦境或者触景生情
+ 对创伤性经历选择性遗忘
+ 持续的回避
+ 持续的警觉性增高
+ 焦虑和抑郁与上述表现相伴随，

## 适应障碍

在重大的生活改变或者应激性生活事件的适应期，出现主观痛苦和情绪紊乱状态，在遭遇事件后1个月起病，病程不超过6个月

+ 抑郁心境、焦虑、烦恼
+ 无力应付的感觉、无从计划或者难以维持现状
+ 一定程度的处理日常能力受损
+ 可伴随品行障碍，尤其是青少年



# 人格障碍

是个体发育过程中，因遗传、先天已经后天不良环境造成的个体心理与行为的持久性的**固定行为模式**， 给个体或者社会造成痛苦

病程开始与童年、青少年、成年早期，现年18以上

## 偏执性人格障碍

以猜疑偏执为主

## 分裂性人格障碍

又称妄想型障碍

行为奇特，情绪冷漠、人际关系明显缺陷

## 反社会性人格障碍

以行为不符合社会规范

## 冲动性人格障碍

又称攻击性人格障碍，以阵发性情绪爆发，

## 表演性人格障碍

以过分感情用事或者夸张言行

## 强迫性人格障碍

过分要求严格与完美无缺

## 焦虑性人格障碍

一贯紧张、提心吊胆

## 依赖性人格障碍

依赖，不独立



## 人格障碍与精神分裂症区别

+ 认知障碍与否
+ 意识障碍与否
+ 法律条文





# 错题

急性应激障碍的病程为数小时至数天

癔症性人格障碍又称 表演性人格障碍

焦虑性人格障碍的特点：总是感到不安、对他人意见过分过敏、习惯性夸大潜在危险

妄想性障碍特点： 妄想内容有一定现实性、病前人格多固执、主观；病程发展缓慢

急性短暂性精神障碍特点：起病前有相关心因性、3个月可以完成恢复、以幻觉妄想症状为主

急性应激障碍表现：意识障碍、人格解体、定向障碍、强烈恐惧
