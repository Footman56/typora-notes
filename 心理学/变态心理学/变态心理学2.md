# 认知障碍

## 感觉障碍

### 感觉过敏

对外界**一般强度**的刺激 感受性增高，感觉阈值降低

多见于 神经症、更年期综合症

### 感觉减退

对外界**一般强度**的刺激 感受性减少，感觉阈值增高

多见于抑郁状态、木僵状态和意识障碍

对外界刺激完成不能感知 称为感觉缺失，多见于癔症

### 内感性不适

躯体内部 **性质不明确**、**部位不具体**的**不舒服感** 或难以忍受的异样感觉

多见于 精神分裂症、抑郁状态、神经症、脑外伤后综合症

## 知觉障碍

### 错觉

对客观事物的歪曲的认识

正常的错觉可以纠正

但是病理性的错觉 不能接受现实的检验，错觉常常有恐怖性质

### 幻觉

无对象性的知觉（感受到事物不是真实存在的）

根据**感受器不同**，分成 幻听、环视、幻嗅、幻味、幻触、内脏性幻觉

根据 **幻觉体验来源不同**，分成 真实性幻觉、假性幻觉

根据 **幻觉产生的特殊条件**，分成 功能性幻听、思维回响【思维鸣响】、心因性幻觉

**幻听是临床最多的表现**

#### 幻听

包括言语性【听到说话】和非言语性【听到自然的声音】

言语性更常见，**对精神疾病的诊断和鉴别诊断具有临床意义**

言语性幻听：

+ 命令性幻听
+ 评论性幻听
+ 争论性幻听

幻听 可见多种精神疾病，精神分裂症、器质性、心因性、功能性精神障碍

#### 幻视

+ 原始性幻觉
+ 同外界事物的形象一样

幻视 见于 精神分裂症、脑器质性疾病和 高热患者

#### 幻嗅

闻到异味

#### 幻味

在食物或者水中 尝到**特殊的味道**

#### 幻触

皮肤或者黏膜上有异常的感觉

#### 内脏幻觉

患者躯体内部 **有性质很明确**、**部位很具体**的异常知觉

#### 真实性幻觉

幻觉 形象 清晰、生动、位置精确 ，【需要借助感受器】

#### 假性幻觉

幻觉形象 模糊、不生动、位置不精确【关闭感受器后仍能感受】

#### 功能性幻听

在某个感受器处于功能活动状态的同时出现幻觉

与正常知觉同时出现、同时存在、同时消失，两者不相容

#### 思维回响【思维鸣响】

患者能**听到自己所思考的内容**

#### 心因性幻觉

**强烈的精神刺激**引发的幻觉

## 感知综合障碍

能感知客观事物，但是对客观事物的某一属性有错误的感知

### 视物变形症

### 非真实感

### 窥镜症

患者认为自己的面孔，体形改变了形状

|              | 实际存在的物体 | 主要特点                 |
| ------------ | -------------- | ------------------------ |
| 幻觉         | 无             | 虚幻的类幻觉、表象体验   |
| 错觉         | 有             | 对本质的错误感知         |
| 感知综合障碍 | 有             | 对**部分属性**的歪曲感知 |

# 思维障碍

## 思维形式障碍

### 思维奔逸

兴奋性的思维联想障碍，指思维活动**量的增加**和思维联想的**速度加快**

严重时 会根据 音韵联想，字意联想，谈话很容易随环境变化而吸引

### 思维迟缓

抑制性的思维联想障碍，思维活动**缓慢** ，语速慢、反应迟钝

### 思维贫乏

患者**思想内容空虚**，概念和**词汇贫乏**，语速不慢 

### 思维松弛【思维散漫】

患者的思维活动 联想松弛、内容散漫 ，思维的**目的性、连贯性、逻辑性** 的障碍 ，表现为 “答非所问”

### 破裂性思维

在意识清楚的情况下，思维联想过程破裂，谈话内容缺乏**连贯性**和**逻辑性**

**是精神分裂症的特征性的思维联想障碍之一**

严重的破裂性思维

+ 思维不连贯 

  在意识障碍下 **语词杂拌**

+ 语词杂拌

  在意识清楚下，语句之间缺乏连贯性，语句片断

### 思维中断

患者**无意识障碍**，无明显的外界干扰，思维过程在**短时间内中断**，有明显的**不自主感**

### 思维插入

思考过程中突然出现与主题无关的内容，像是**别人强加**的

### 思维被夺

患者在思考过程中 一些思想被**外界力量夺走**

### 思维云集【强制性思维】

不受患者意愿支配的思想，强制性的大量涌现，内容杂乱多变、突然出现、突然消失



思维插入、思维被夺时患者还是有自己的意识的，思维云集就完全不受自己意愿支配

### 病理性赘述

谈话过程中夹杂不必要的细节，不觉的啰嗦，认为有必要

### 病理性象征性思维

患者主动以一些普通的概念、词语、动作表达特殊的、不经患者解释他人无法理解的含义。会把象征的东西当作现实的东西

### 语词新意

创造一些文字、图形 赋予特殊的意义

### 逻辑倒错性思维

以思维联想过程中逻辑性的明显障碍为主要特征。逻辑推断是错误的

| 思维贫乏 | 脑子空虚，没有什么可以想的 |
| -------- | -------------------------- |
| 思维迟缓 | 反应迟钝                   |



|            | 特征     | 自我感觉 | 给人印象 |
| ---------- | -------- | -------- | -------- |
| 思维奔逸   | 思潮澎湃 | 脑子灵活 | 夸夸其谈 |
| 病理性赘述 | 节外生枝 | 难以表达 | 拖泥带水 |



|          | 主要特点 | 表现形式                           | 临床意义       |
| -------- | -------- | ---------------------------------- | -------------- |
| 思维散漫 | 结构松散 | 句与句之间的联系，整段内容缺乏中心 | 早期精分       |
| 思维破裂 | 联想断裂 | 句与句间无联系，只有词之间的联系   | 精分特征性症状 |



## 妄想

是一种脱离实际的病理性思维

特征：毫无根据、坚信不疑、与我有关

根据**内容划分**：被害妄想、关系妄想、特殊意义妄想、物理影响妄想、夸大妄想、自罪妄想、疑病妄想、嫉妒妄想、钟情妄想、内心被揭露感

根据 起源以及妄想与其他精神症状的关系 ：原发性妄想、继发性（在错觉、幻觉、逻辑推理障碍为基础产生的妄想）



### 关系妄想

实际无关的事。认为与他有关

### 被害妄想

### 特殊意义妄想

周围人的言行、举动，与自己有关，并且有特殊意义

### 物理影响妄想

+ 影响妄想： 外界力量影响思维
+ 物理影响妄想：先进仪器的影响

### 夸大妄想

### 自罪妄想-罪恶妄想

毫无根据的认为自己有罪

### 疑病妄想

### 嫉妒妄想

坚信配偶对其不忠

### 钟情妄想（被钟情妄想）

### 内心被揭露感（被洞悉感）

认为内心的想法或者本人的情况 未经语言表达 ，别人就知道

产生的原因：

+ 在其他症状基础之上



### 原发性妄想

突然出现、内容不可理解 ，**最为常见**

#### 妄想知觉

对正常知觉赋予妄想性意义，一个事物有别的含义

### 妄想心境

对熟悉的环境感到气氛不对

**原发性妄想 是精神分裂症的 特征性症状**

## 思想内容障碍

### 强迫观念【强迫性思维】

某一种观念 反复出现 在脑海， 认为想法是不必要的，想摆脱但摆脱不了 ，**感到痛苦**

+ 强迫性回忆
+ 强迫性穷思竭虑
+ 强迫性计数
+ 强迫性怀疑
+ 强迫性对立观念



|            | 内容               | 持续                 | 主观体验           | 摆脱愿望 | 自知力 |
| ---------- | ------------------ | -------------------- | ------------------ | -------- | ------ |
| 强迫性思维 | 无法摆脱的苦思冥想 | 重复固定             | 焦虑苦闷           | 强烈     | 完全   |
| 强制性思维 | 不由自主的外来思想 | 变化不定，持续时间短 | 偶发短暂、陌生意外 | 不强烈   | 不全   |

意识中占主导地位的错误观念、其发生有事实根据、有强烈的感情色彩。
