# 初步接待

初步接待中，心理咨询师的工作主要是提供给一个让求助者释放压抑的空间，不在于解决求助者的困扰

## 操作步骤

1. 提前做好咨询前的准备工作

   + 配置合理咨询场所：私密性、安全、
   + 注意仪容
   + 保持社交距离
   + 与来访者保持90度相坐，准备水、纸巾、挂钟

2. 礼貌接待方式和礼貌语言

3. 间接询问求助者希望得到哪些方面的帮助，不可直接逼问

4. 询问结束后，明确表明态度

   是否能向求助者提供帮助

5. 向求助者说明保密原则

6. 向求助者说明心理咨询的性质

   ```
   在向求助者表明可以对他提供心理学帮助后，应立即简约地向求助者说明心理咨询的性质。确保求助者了解什么是心理咨询，心理咨询如何进行，心理咨询主要解决什么问题，而不能解决什么问题等等。 “协助”
   ```

7. 说明求助者的权利和义务

8. 与求助者协商，使用哪种咨询方式

   面谈效果最好，视频次之



## 注意事项

+ 避免紧张情绪
+ 语言表达：语速适中，吐字清楚
+ 说明保密原则：对心理咨询中的保密原则
+ 说明心理测量功能的有限性：心理测量、心理咨询的功能都是有限 的，咨询人员不可为了获取求助者的信任而随意夸大它们的功能
+ 咨询时的仪态：交通中不能东张西望，应注意力集中，认真倾听或发问

# 摄入性会谈

摄入性会谈是：以获取信息为目的的会谈

## 操作步骤

1. 确定谈话的目标、内容与范围
   + 求助者主动提出的求助内容
   + 初次接待中观察到的疑点
   + 依据心理测评结果的初步分析发现问题
   + 上级心理咨询师为进一步诊断而下达的谈话目标

2. 确定提问方式

   一般情况下，应使用开放式提问，不使用封闭式提问。主要依据谈话目标、内容来确定。

   封闭性谈话适合缩小谈话范围

3. 倾听

   + 全神贯注、倾心地听
   + 在听的过程中，不要随便打断求助者谈话并插入自己的谈话内容的评价
   + 在听的过程中，最重要的是及时把握关键点
   + 在倾听的时候，还有思考，要及时而迅速地判断求助者的谈话是否合乎常理

4. 把握谈话方向

   会谈的方向、所涉及的问题及会谈的时间都必须是有计划、有目的的。控制会谈和转换话题的技巧很多，而且可以随机应变。最常用的方法有

   + 释义：即征得来访者的同意后，把来访者的话重复一下并做解释，之后，立即顺便提出另一个问题
   + 中断：礼貌中段重复的问题
   + 使用情感的反射作用：有意识地激一下来访者，使他把谈话转向某类问题
   + 引导：即由目前话题引向另一话题

#关键技术

##  倾听技术

倾听是心理咨询的第一步，是建立良好的咨询关系的基本要求，是每个心理咨询员的基本功。其作用在于:

1、可表达对求助者的尊重; 

2、可充分了解求助者的各方面情况;

3、可使对方在比较宽松和信任的氛围下倾诉

## 态度

心理咨询师的态度：**非批判性的态度**

中性：理解、接受



## 区别谈话内容

对来访者的谈话内容进行区别和鉴别十分重要

区别 谈话内容的程度

鉴别 内容的真伪

## 谈话的种类

+ 摄入性谈话：收集资料 【最常用】
+ 鉴别性谈话：确定使用什么测验和鉴别测验
+ 治疗性谈话：针对精神变态和行为异常
+ 咨询性谈话：涉及**健康人**的某些问题
+ 危机性谈话：当事人遇到意外时

## 怎么提问题

封闭性提问：

通常使用“是不是”、对不对、要不要 ，而回答也是 “是“，“否”式简单答案

开放性提问：“什么”、“如何”、“怎么”、“能不能”、“愿不愿意”等词来发问，让来访者就有关问题、思想、情感给予详细的说明

在咨询师还没有真正了解来访者时，或还没有掌握语言交流技巧时，最有帮助的办法是 开放性提问



谈话过程中，不易提问过度，会产生以下消极作用

+ 造成依赖
+ 责任转移
+ 减少当事人的自我探索
+ 产生不准确的信息
+ 当事人处于被审问地位而产生防卫心理和行为
+ 影响交谈中必要的概括和说明

凯利把临床交谈提问的性质归一下几类

+ “为什么...  ” 这样的问题，有强烈的暗示性，可以改为 “怎样”，“什么”的形式
+ 多重选择性问题：应去掉选择部分【不要给咨询者暗示，限制】
+ 多重问题：避免
+ 修饰性反问：避免
+ 责备性问题：杜绝
+ 解释性问题：杜绝【 让咨询者自己思考】

## 选择谈话内容的原则

+ 可接受：适合患者的接受能力，符合患者的兴趣
+ 有效
  + 对患者的病因有直接或间接的针对性
  + 对患者的个性发展或矫正起关键作用
  + 对深入探索患者的深层病因有意义
  + 对患者症状的鉴别诊断有意义
  + 对帮助患者改善认知和正确理解问题有帮助
+ 积极：对改变患者的态度有积极作用

注意事项：

+ 态度中性
+ 提问避免失误
+ 在摄入性谈话中，除提问和引导性语言之外，不能讲任何题外话
+ 不要指责、批判性语言阻止或者扭转求助者的谈话内容
+ 在摄入性谈话后不应该给出决定性结论
+ 结束语要诚恳、客气，不能用生硬的话做结束语



# 正确使用心理测验

心理测量按测量的目的分为：

+ 智力测量
+ 人格测量
+ 心理与行动问题评估
+ 应激与相关问题评估

注意实现：

不要乱用心理测验，不得使用 “地毯式轰炸”方式，不要过分依赖测验（测验有主观性）

# 一般临床资料整理

摄入性会谈获取临床资料-》临床资料进行整理、分析、综合-〉作为诊断依据-》完成诊断

## 对临床资料的归类、解释、验证

+ 当事人个体方面的情况
+ 有关当事人的环境条件
+ 别人对他的评价

## 资料的可靠性

验证资料的可靠性：

+ 补充提问
+ 使用问卷和心理测验的方法
+ 比较同一资料的不同来源

## 资料的意义

收集完资料之后，要赋予资料意义

+ 就事论事
+ 寻找相关
+ 迹象分析

## 影响资料可靠性

+ 暗示：过分随意地交谈、咨询师的倾向性

+ 咨询师的早期印象

+ 求助者处境和人格特点

  

# 初步诊断

1. **确定造成求助者心理与行为问题的关键点**
2. 确定求助者的问题是否属于咨询范围
3. 对求助者形成初步印象，对一般心理健康水平 进行分析
4. 一般心理问题的诊断
5. 严重心理问题的诊断
6. 提出心理评估报告



初步印象，是指心理咨询师对资料整理分析之后，必须对求助者的心理问题和行为问题就严重程度和归类诊断方面，形成大致的判断。

1. 首先通过三原则确实 是正常还是异常心理 【异常心理不是工作范围】
2. 正常心理之后使用 常性冲突还是变性冲突
3. 如果是变性冲突的话，使用 **许又新**的评分量表确定是 可疑神经症、神经质
4. 如果是常性冲突的话，比较时间，刺激程度，是否泛华来确定是 一般心理问题、严重心理问题

### 三原则

+ 主观世界与客观世界的统一性

+ 心理活动的内在一致性

  精神活动虽然可以分为认知、情绪情感、意志行为等部分，但它自身确是一个完整的统一体，各种心理过程之间具有协调一致的关系

+ 人格的相对稳定性

确定工作范围

+ 明确自己的胜任力。
+ 理论上，健康心理咨询的主要对象是一般心理问题、严重心理问题和部分神经症性问题
+ 对精神病性问题，心理咨询师只能进行有条件的辅助性 工作 
+ 慎重对待出现的神经症性问题的求助者和能够确诊为神 经症的求助者
+ 综合分析和鉴别精神病性问题和神经症性问题 。

## 变性与常性冲突

常性冲突：

+ 与现实处境有直接联系
+ 带有明显的道德性质

变性冲突：

+ 与现实处境没有关系
+ 不带明显的道德色彩

### 神经症与正常鉴别的量化评分

**首先需要确定是变性冲突**

+ 病程
  + 1分：不到3个月
  + 2分：3月～1年
  + 3分：1年以上
+ 精神痛苦程度
  + 1分：自己可以摆脱
  + 2分：须靠别人帮助或改变处境才能摆脱
  + 3分：几乎完全无法摆脱
+ 社会功能
  + 1分：轻微妨碍，效率下降
  + 2分：不得不减轻工作或改变工作，或某些社交场合不得不尽量避免
  + 3分：完全不能工作，病休，或某些社交场合完全回避

结论： <3 分不能诊断神经症，4-5分可疑神经症，>=6分 可以诊断神经症



可疑神经症

心理不健康状态，已接近神经衰弱或神经症，或者它本身就是神经衰弱或神经症的早期阶段。



神经症

神经症是一种精神障碍，主要表现为持久的心理冲突，病人觉察到或体验到这种冲突并因之而深感痛苦且妨碍心理功能或社会功能，但没有任何可证实的器质性病理基础

+ 意识的心理冲突
+ 精神痛苦
+ 持久性
+ 神经症妨碍着病人的心理功能或者社会功能
+ 没有任何器质性病变为基础



## 一般和严重心理问题

一般心理问题

+ 由于现实生活、工作压力、处事失误等因素而产生内心冲突， 并因此而体验到不良情绪(如厌烦、后悔、懊丧、自责等
+ 不良情绪不间断地持续满一个月，或不良情绪间断地持续两个 月仍不能自行化解;
+ 不良情绪反应仍在相当程度地理智控制下，始终能保持行为不 失常态，基本维持正常学习、生活交往，但效率有所下降;
+ 不良情绪的激发因素仅仅局限于最初事件

严重心理问题

+ 引起“严重心理问题”的原因，是较为强烈的、对个体 威胁较大的现实刺激
+ 从产生痛苦情绪开始，痛苦情绪间断或不间断地持续时 间在两个月以上，半年以下
+ 遭受的刺激强度越大，反应越强烈。
+ 痛苦情绪不但能被最初的刺激引起，而且与最初 刺激相类似，相关联的刺激，也可以引起此类痛苦， 即反应对象被泛化
