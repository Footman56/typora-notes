# 建立咨询关系

建立良好的咨询关系时心理咨询的核心内容

## 尊重

+ 尊重意味着咨询师无条件接纳求助者
+ 尊重意味着平等
+ 尊重意味着礼貌
+ 尊重意味着信任
+ 尊重意味着保护隐私
+ 尊重意味着真诚

## 热情

有浓厚的感情色彩

+ 在初诊接待阶段打好热情的基础

+ 通过倾听和非言语行为，表达热情
+ 咨询中认真、耐心、不厌其烦是热情最好表达
+ 咨询结束时时求助者感受到温暖

## 真诚

+ 真诚不等于实话实说【有目的】
+ 真诚不是自我发泄
+ 真诚应实事求是
+ 表达真诚应该适度
+ 真诚还体现非言语交流上
+ 表达真诚应考虑时间因素

## 共情

+ 咨询师应从求助者的而不是自己的角度看待问题
+ 共情不是要求与求助者有相似的经历感受，而是设身处地的理解
+ 表达共情因人而异
+ 共情应适度
+ 共情应善于使用躯体语言
+ 咨询师应验证自己是否与求助者产生共情





# 参与性技术

## 倾听技术

+ 不要打断求助者，做评判
+ 不要急于下结论
+ 不要轻视求助者
+ 干扰、转移求助者的话题
+ 不要询问过多，概括过多，可问可不问时不问，可说可不说时不说
+ 多给予适当的鼓励性回应

## 封闭式、开发式提问

## 鼓励技术

鼓励别人讲下去；可重复别人的话，或者使用“讲下去”，还有吗

## 重复技术

重复别人说过的话，在出现不合理，与常理不符的时候使用
