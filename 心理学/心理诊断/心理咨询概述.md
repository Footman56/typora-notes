心理咨询 VS 心理辅导

心理咨询：帮助他人做出改变、平等的工作同盟

心理辅导：帮助他人做出决策，师生关系



# 学术背景

高尔顿：1882用测量的方法对心理活动个别差异 的研究

卡特尔：1890发表《心理测验与测量》

伟特默：1896宾夕法尼亚大学开办儿童行为矫正诊所

比内与西蒙：1904帮助弱智儿童编制智力 测量表

罗杰斯：心理咨询是通过与个体持续的、直接的接触，向其提供心 理帮助并力图促进其行为、态度发生变化的过程

心理咨询师是**运用心理学以及相关知识**，**遵循心理学原则**，**通过心理咨询的**技术和方法，帮助求助者解除心理问题的专业人员

心理咨询师应有的思维方式和态度

+ 唯物主义、科学原则

+ 普通联系观点

  + 身心一体
  + 心神社会交换
  + 整体性

+ 限制性观点

  + 责任：对咨询者负责、需咨询者努力、聚焦心理问题
  + 时间：50分钟，1周1次
  + 感情：以求助者成长为目的
  + 目标：心理问题、解决程度

+ 历史逻辑现实相统一发展观

+ 中立性态度

# 心理咨询的对象

发展性咨询：精神正常，但遇到了心理相关的现实问题

心理健康咨询：精神正常，但心理健康出现障碍并请求帮助

鉴别诊断：特殊对象，临床治愈或者潜伏期的精神病患者



# 心理咨询师的任务

1. 认识自己的内外世界

   侧重对内部世界的认识和评价

2. 了解和改变不合理的观念

3. 学会面对现实和应对现实

   + 感性反应：对外部事物的情绪化应对
   + 理性反应：用概念和事物间的客观逻辑去反应外部事物
   + 悟性反应：是积极面对的心态

4. 使求助者学会理解他人

5. 使求助者更好的认识自己

6. 协助求助者建构合理的行为模式

## 分析

排序：先后顺序

筛选：剔除无关

比较：确定主次

分析：寻找原因

找出造成问题的主因和诱因

## 评估

+ 将获得的资料进行分析比较
+ 对主因、诱因与临床症状的因果关系进行解释
+ 确定心理问题的由来、性质、严重程度
+ 确定其在症状分类中的位置

## 诊断、鉴别诊断

- 症状定性:按症状的表现确定其性质
- 症状区分:将已经定性的症状和在现象上与其相近、性质相类似的其他症状做细致的区分，并做出明确判断
- 症状确定:确定鉴别诊断的关键症状和特征
- 症状诊断:按现行的症状诊断标准，进行鉴别诊断

## 精神分析理论

分区观点：意识、潜意识、前意识

结构观点：自我、本我、超我

动力学观点： 力比多-》种族延续，影响本能=〉个体保存

发展观点：

+ 口唇期：0～1岁
+ 肛欲期：1～3岁
+ 前生殖器阶段 3～6岁
+ 潜伏期 6～11，12
+ 生殖器 11，12之后

适应观点：人的本能得以实现，必须经过不懈的努力和艰苦的应对——适应

当自我把焦虑当成一种危险或不愉快的信号时，它就会做出反应，形成“自我防御机制”，就是“自我”在承受“本我”的欲望压力时，同时又顾及现实要求的压力。
 • **自我防御机制:压抑、投射、反向、合理化、升华、转移**等。

### 心理动力学取向

+ 使潜意识意识化
  + 禁闭的思想和情感是有害的，释放它们可以起到疏导作用 
  + 不将潜意识的内容通过会谈引导到意识之中，潜意识的力量会逐渐变得庞大而邪恶 
  + 支配我们的如果是潜意识的，那么我们就无法控制它们 
  + 词语是潜意识与有意识心理的桥梁和纽带，非线性非言语-线性言语性

+ 支持薄弱的自我功能

  + 由于**自我需要调节本我和超我的关系**，所以大部分心理能量都消耗在**控制和压制**之上。 
  + 弗洛伊德认为只有这三个我和睦相处，人才能健康发展。如果发生冲突，长期下去则会导 致心理问题甚至精神疾病的发生。 
  + 所以**自我起着关键作用**。通过支持薄弱的自我功能，可以有效的帮助来访者。

+ 使发展再现生机

  精神分析重视童年经历，早期的创伤会导致成年后的一些心理问题， 所以让来访者再次发展，能够帮助其走出创伤。可能发生的		新进展包括: 

  + 发展出**思考自身和调控自尊的新方式**
  + 发展出**与他人交往的新方式**
  + 发展出**更灵活、更具适应的应对机制**

## 认知心理学理论

认知:指一个人对某一事件 的认识和看法。(包括对过 去事件的**评价**;对当前事件 的**解释**;对未来事件的**预期**

**目的:**不仅是针对行为、情绪的外在表现，而且分析人 的现实思维活动，找出错误认知。通过**改变人的认知过 程来纠正人适应不良的情绪和行为**

基本观点：

+ 认知过程是行为和情感的中介
+ 适应不良行为与适应不良认知有关
+ 治疗者的任务是找出不良和歪曲的认知，提供学习或者训练的方法，矫正其思维方式使之合于理性

采用认知行为疗法

认知行为治疗的基本假设包括：

1. 人们的思想和信念对他们适应不良行为有作用
2. 通过改变人的信念、态度、思维便能改变他们的适应不良行为
