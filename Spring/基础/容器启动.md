# 容器创建

```java
AnnotationConfigApplicationContext context = new AnnotationConfigApplicationContext(BeanConfig.class);

public AnnotationConfigApplicationContext(Class<?>... componentClasses) {
		this();
		register(componentClasses);
		refresh();
	}
```

# 配置扫描

配置类的是由BeanFactoryPostProcessor来实现的，BeanFactoryPostProcessor 的作用就是BeanFactory 的后置处理器。BeanFactoryPostProcessor只能修改BeanDefinition

BeanDefinitionRegistryPostProcessor 可以新增BeanDefinition。就可以新增一些系统默认的BeanDefinition ，有了这些默认的BeanDefinition就可以做相应的功能啦
