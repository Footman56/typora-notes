需求：http 接口, 属性中有一个是集合，集合里面的元素都有同一父类，想要根据不同的类型获取到不同的子类，属于【多态反序列化】

解决方法，添加 `@JsonTypeInfo` `@JsonSubTypes` 两个注解，思路是根据父类的属性来确定反序列化的子类

需要特殊注意的是

+ @JsonTypeInfo 中的property 为父类的属性
+ @JsonSubTypes 指定子类，name为inspectionStatus 属性的值，value 指定子类

```java
@JsonTypeInfo(use = JsonTypeInfo.Id.NAME, include = JsonTypeInfo.As.EXISTING_PROPERTY, visible = true, property = "inspectionStatus")
@JsonSubTypes({
        @JsonSubTypes.Type(value = KpiTargetRegistrationModel.class, name = "0"),
        @JsonSubTypes.Type(value = KpiTargetConfirmedModel.class, name = "1"),
        @JsonSubTypes.Type(value = KpiCompleteValueEntryModel.class, name = "8"),
        @JsonSubTypes.Type(value = KpiSelfEvaluationSettingModel.class, name = "2"),
        @JsonSubTypes.Type(value = KpiExaminingSettingModel.class, name = "3"),
        @JsonSubTypes.Type(value = KpiPerformanceCalibrationModel.class, name = "4"),
        @JsonSubTypes.Type(value = KpiPerformanceCommunicationModel.class, name = "5"),
        @JsonSubTypes.Type(value = KpiResultsPublicSettingModel.class, name = "6"),
        @JsonSubTypes.Type(value = KpiFlowCustomizeModel.class, name = "9"),
})
```
