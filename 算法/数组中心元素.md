问题：给定一个整数数组 nums，请编写一个能够返回数组
“中心下标”的方法。
中心下标是数组的一个下标，其左侧所有元素相加的和等于右侧所有元素相加的和。
如果数组不存在中心下标，返回-1。如果数组有多个中心下标，应该返回最靠近左边的
那一个。
注意：中心下标可能出现在数组的两端。

**技巧在于找到对应的规律，可以用等式来表达的，用这个等式来转换程序。**

# 解决一：

左边的元素和=右变的元素和，使用一个指标记录左边的元素和，用总和减去左边元素和就是右边的元素和

```java

    public static int findCenterIndex(int[] nums) {
        // 1. 计算数组的总和
        int totalSum = Arrays.stream(nums).sum();
        // 2. 定义一个变量来记录左侧的和
        int leftSum = 0;
        // 3. 遍历数组，计算每个位置的右侧和，并比较左右两侧的和
        for (int i = 0; i < nums.length; i++) {
            // 右侧和 = 总和 - 左侧和 - 当前元素
            int rightSum = totalSum - leftSum - nums[i];

            // 如果左右两侧的和相等，返回当前索引
            if (leftSum == rightSum) {
                return i;
            }
            // 更新左侧的和
            leftSum += nums[i];
        }
        // 如果没有找到中心索引，返回-1
        return -1;
    }


    public static int findCenterIndex2(int[] nums) {
        // 1. 计算数组的总和
        int totalSum = Arrays.stream(nums).sum();

        // 2. 定义一个变量来记录左侧的和
        int leftSum = 0;

        // 3. 遍历数组，计算每个位置的右侧和，并比较左右两侧的和
        for (int i = 0; i < nums.length; i++) {
            // 右侧和 = 总和 - 左侧和 - 当前元素
            // 更新左侧的和

            int rightSum = totalSum - nums[i];

            // 如果左右两侧的和相等，返回当前索引
            if (2 * leftSum == rightSum) {
                return i;
            }

            leftSum += nums[i];


        }

        // 如果没有找到中心索引，返回-1
        return -1;
    }

    public static void main(String[] args) {

        int centerIndex = findCenterIndex2(new int[]{1, 7, 3, 6, 5, 6});
        System.out.println(centerIndex);


    }
```
