题目：统计100以内的所有素数

采用埃筛法，思路是借助一个数组去维护数据是否满足，如果满足就有一个标志位，需要通过一定的技巧去缩小遍历的次数。去除掉一些重复的条件。

```java
 /**
     * 通过一个数组来设置标记位，统计标记位的数量就是结果，如果一个数能由其他的数相乘，那么这个数就不是素数，就标记为false,那么相关的数据就都不是
     * @param n
     * @return
     */
    public static int countPrime(int n) {
        // 1. 创建一个布尔数组，默认值为false，表示所有数都是素数
        boolean[] isPrime = new boolean[n + 1];


        // 2. 从2开始，依次标记每个素数的倍数为非素数，遍历时只需遍历到 根号n就行,因为如果一个数能被大于根号n的数整除，那么它必然能被小于根号n的数整除。
        for (int i = 2; i * i <= n; i++) {
            if (!isPrime[i]) {
                // 不是素数的话就符合条件
                for (int j = i * i; j <= n; j += i) {
                    // 如果i是素数，那么2*i，3*i，4*i...都不是素数，标记为true，但是这样会有重复标记的情况，比如6=2*3，6= 3*2，所以需要从i*i开始标记，这样就不会有重复标记的情况了。
                    isPrime[j] = true;
                }
            }
        }

        int count = 0;
        // 3. 统计素数的数量
        for (int i = 2; i <= n; i++) {
            if (!isPrime[i]) {
                count++;
            }
        }
        return count;
    }

    public static void main(String[] args) {
        int i = countPrime(100);
        System.out.println(i);

    }
```
