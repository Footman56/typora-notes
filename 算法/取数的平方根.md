# 思路一：二分查找法

```java
 /**
     * 二分查找法，通过用于搜索在区间内查找有效的值。
     * @param x
     * @return
     */
    public static int binarySearch(int x) {
        // 确定左右区间和结果值
        int left = 1;
        int right = x;
        int index = -1;

        // 循环条件，左小于右
        while (left <= right) {
            // 取中间的值
            int mid = left + (right - left) / 2;

            if (mi d * mid <= x) {
                // 左侧不符合，取右半区间
                left = mid + 1;
                index = mid;
            } else {
                // 右侧不符合，取左半区间
                right = mid - 1;
            }
        }
        return index;
    }

    public static void main(String[] args) {

        int binarySearch = binarySearch(26);

    }
```

# 思路二：牛顿迭代

核心思想是：用函数在当前点的切线去逼近函数的根。转换成数学的函数求解问题。借助牛顿迭代一点一点的逼近

### 几何直观

1. 先随便猜一个初始值 x₀
2. 在 (x₀, f(x₀)) 处作切线
3. 切线和 x 轴的交点作为新的近似值 x₁
4. 不断重复这个过程

假设  x* x = a ,那么下一个x(n+1) 带入迭代公式 得出 x(n+1)  =  （xn + a / xn） /2  ，用这个函数不断去逼近

```java

    public static int newton(int x) {
        if (x == 0) {
            return 0;
        }
        // 初始值可以随便设置，设置为x就行了
        double sqrt = sqrt(x, x);
        return (int) sqrt;
    }


    /**
     *
     * @param x 平方根
     * @param i 待求的的数
     * @return
     */
    public static double sqrt(double x, int i) {
        if (x == 0) {
            return 0;
        }
        double result = (x + i / x) / 2;
        // 结束条件
        if (result == x) {
            return result;
        }
        // 通过递归不断的求解，直到结果不再变化为止
        return sqrt(result, i);
    }
```
