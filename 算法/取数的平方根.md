题目：获取x的平方根，不要使用sqrt函数

# 思路一：二分查找法

```java
 /**
     * 二分查找法，通过用于搜索在区间内查找有效的值。
     * @param x
     * @return
     */
    public static int binarySearch(int x) {
        // 确定左右区间和结果值
        int left = 1;
        int right = x;
        int index = -1;

        // 循环条件，左小于右
        while (left <= right) {
            // 取中间的值
            int mid = left + (right - left) / 2;

            if (mi d * mid <= x) {
                // 左侧不符合，取右半区间
                left = mid + 1;
                index = mid;
            } else {
                // 右侧不符合，取左半区间
                right = mid - 1;
            }
        }
        return index;
    }

    public static void main(String[] args) {

        int binarySearch = binarySearch(26);

        System.out.println("binarySearch = " + binarySearch);
    }
```
