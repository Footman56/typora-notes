问题：一个有序数组 nums，原地删除重复出现的元素，使每
个元素只出现一次，返回删除后数组的新长度。
不能使用额外的数组空间，必须在原地修改输入数组并在使用0（1）额外
空间的条件下完成。
例：输入：［0,1,2,2,3,3,4］
输出：5



# 解决思路：双指针

一个慢指针、一个快指针。慢指针指向的是最后一个重复的元素，快指针是遍历指针。如果找到不一样的元素后，就将快指针的值赋值给慢指针下一个位置

可以利用多个指针来遍历数组，一个指针用于做遍历，另一个指针来做业务。适合情景为空间要求较高

```java
public static int removeDuplicates(int[] nums) {
        // 1. 定义一个指针，指向当前不重复的元素的最后一个位置
        int i = 0;
        // 2. 遍历数组，如果当前元素和前一个元素不相同，那么就将当前元素放到指针的下一个位置，并将指针向后移动一位
        for (int j = 1; j < nums.length; j++) {
            if (nums[j] != nums[i]) {
                i++;
                nums[i] = nums[j];
            }
        }
        // 3. 最后返回指针的位置加一就是不重复元素的数量
        return i + 1;
    }

    public static void main(String[] args) {
        int removeDuplicates = removeDuplicates(new int[]{1, 1, 2, 3, 4, 4, 6, 6, 7, 8, 9, 9});
        System.out.println(removeDuplicates);
    }
```
