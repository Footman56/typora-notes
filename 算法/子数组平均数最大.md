问题：子数组最大平均数 给一个整数数组，找出平均数最大且长度为k的下标连续的子
数组，并输出该最大平均数。输入：［1,12，-5，-6,50,3］，k=4
输出：12.75 最大平均数（12-5-6+50）/4 =51/4 =12.75

采用滑动窗口算法，窗口的宽固定，移动的时候新增一个数的同时减少一个数

```java
 public static double maxAverage(int[] nums, int k) {
        // 1. 计算前k个数的和
        int sum = 0;
        for (int i = 0; i < k; i++) {
            sum += nums[i];
        }

        int max = sum;

        // 2. 从第k个数开始，新增一个数的同时减少一个数,窗口向右滑动
        for (int i = k; i < nums.length; i++) {
            sum += nums[i] - nums[i - k];
            max = Math.max(sum, max);
        }

        return (double) max / k;
    }
```
