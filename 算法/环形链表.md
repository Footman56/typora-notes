问题：给定一个链表，判断链表中是否有环。如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达该节点，则链表中存在环

**技巧：将对应的问题转换成特殊的数据结构来解决。本质就是检查是否有重复的元素。**

# 思路一： set去重

```java
 public static boolean hasCycle1(LinkedNodeReversal.LinkedNode head) {

        // 因为是环形结构，那么就意味着有相同的元素，可以采用向set中存放数据来检查是否有重复的
        if (null == head) {
            return false;
        }

        Set<LinkedNodeReversal.LinkedNode> set = new HashSet<>();
        while (null != head.getNext()) {
            if (!set.add(head)) {
                // 不能向集合中添加元素，说明已经存在了，那么就说明有环
                return true;
            }
            // 循环条件
            head = head.getNext();
        }
        return false;
    }
```



# 思路二：快慢指针

如果有环的话，那么快指针一定能与慢指针相遇

```java
public static boolean hasCycle(LinkedNodeReversal.LinkedNode head) {
        if (null == head) {
            return false;
        }
        // 需要使用快慢指针来判断链表是否有环
        LinkedNodeReversal.LinkedNode slow = head;
        LinkedNodeReversal.LinkedNode fast = head.getNext();
        if (fast == null || fast.getNext() == null) {
            return false;
        }

        while (fast != null && fast.getNext() != null) {
            slow = slow.getNext();
            fast = fast.getNext().getNext();

            if (slow == fast) {
                return true;
            }
        }
        return false;
    }
```
