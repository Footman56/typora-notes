问题：求斐波拉切数列中某个位置的数值

# 思路一：暴力递归

```java
  /**
     * 暴力递归法，时间复杂度为O(2^n)，空间复杂度为O(n)
     * @param n
     * @return
     */
    public  static  int calculate(int n) {
        if (n <= 0) {
            return 0;
        }
        if (n == 1) {
            return 1;
        }
        return calculate(n - 1) + calculate(n - 2);
    }
```

# 思路二：去重递归

```java
 /**
     * 去重迭代法，时间复杂度为O(2^n)，空间复杂度为O(n)
     * @param n
     * @return
     */
    public  static  int calculate1(int n) {
        // 用数值来存储已经计算过的结果，避免重复计算
        int[] arr = new int[n + 1];
        return fb(arr, n);
    }

    public static int fb(int[] arr, int n) {
        if (n <= 0) {
            return 0;
        }
        if (n == 1) {
            return 1;
        }
        // 已经算出了结果就使用结果
        if (arr[n] != 0) {
            return arr[n];
        }
        arr[n] = fb(arr, n - 1) + fb(arr, n - 2);
        return arr[n];
    }
```

#  思路三：双指针迭代法

这两个指针可能是左右，前后，快慢。就是迭代中的两个状态

```java
 /**
     * 双指针迭代法，时间复杂度为O(n)，空间复杂度为O(1)，对去重迭代的进一步优化，不需要特别多的参数。
     * 在迭代的过程中，不断的修改指针的值，来达到最后的状态。
     * @param n
     * @return
     */
    public static int calculate2(int n) {
        if (n <= 0) {
            return 0;
        }
        if (n == 1) {
            return 1;
        }
        // 用两个变量来存储前两个数的值，避免重复计算
        int a = 0;
        int b = 1;
        int c = 0;
        for (int i = 2; i <= n; i++) {
            c = a + b;
            a = b;
            b = c;
        }
        return c;
    }
```
